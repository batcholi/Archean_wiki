# Description
La High Voltage Junction permite distribuir energía a 4 puertos para alimentar múltiples componentes desde una sola fuente de energía.

> - La High Voltage Junction no permite combinar energía usándola en sentido inverso, funciona en una sola dirección.
> - La energía disponible se distribuye dinámicamente según la demanda, permitiendo encadenar junctions libremente para alimentar componentes según sea necesario.
> - Puedes monitorizar el consumo de energía de cada puerto así como la energía de entrada desde la ventana de información accesible con la tecla `V`.
