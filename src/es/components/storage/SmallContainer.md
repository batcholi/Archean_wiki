# Description
El Small Container es un componente de almacenamiento con una capacidad de 25 espacios.
Está equipado con dos puertos para conectar conductos de objetos para recibir o enviar objetos.

El puerto de datos permite obtener el contenido del contenedor como una cadena usando el [sistema key-value](/xenoncode/documentation.md#key-value-objects).

>- *El contenedor no tiene la capacidad de extraer o empujar objetos directamente desde sus puertos, es únicamente un componente de almacenamiento.*
