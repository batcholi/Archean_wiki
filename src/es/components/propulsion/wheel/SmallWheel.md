# Description
La Small Wheel es una rueda simple que solo puede frenar y girar, y no tiene suspensión.
No requiere energía para funcionar.
# Usage
La rueda puede conectarse a un ordenador u otro dispositivo para su control.

### List of inputs
|Channel|Function|Range|
|---|---|---|
|0|Brake|0.0 to 1.0|
|1|Steer|-1.0 to +1.0|
