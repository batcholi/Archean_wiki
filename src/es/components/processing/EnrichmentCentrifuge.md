# Description
El componente Enrichment Centrifuge separa una mezcla de moléculas en dos flujos distintos según su masa molecular.

# Usage
Requiere:
- Una alimentación eléctrica continua de 1.000 vatios.
- Una entrada de datos para activarlo.

El fluido se inyecta a través de un puerto situado en la parte superior del componente. En la parte inferior, las moléculas más ligeras salen por un puerto situado más arriba, mientras que las moléculas más pesadas salen por un puerto inferior directamente debajo.

Ambos puertos de salida deben estar correctamente conectados para garantizar un funcionamiento correcto.
