# Description
El Data Bridge es un componente que simplemente permite reubicar un punto final de datos a otra posición.

> El Data Bridge literalmente reubica el punto final de datos al propio Data Bridge.
>
> Para referenciar dichos datos, el Data Bridge debe ser referenciado por alias en lugar del componente del que se intenta leer.
