# Description
El Mini Router es una variante más pequeña del [Router](Router.md) que solo tiene 14 puertos de datos.

Requiere una fuente de alimentación de bajo voltaje para funcionar y consume 50 vatios.

Al igual que el Router estándar, permite asignar alias a los componentes y puede encadenarse con otros Routers para expandir la red.

> El Mini Router que se comunica directamente con un Computer debe estar alimentado. Los demás Routers en la cadena no requieren energía y pueden usarse como un [Data Bridge](DataBridge.md) manteniendo la capacidad de resolver alias y referencias de pantalla.
