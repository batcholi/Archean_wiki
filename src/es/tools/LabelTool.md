# Description
El Label Tool es una herramienta para colocar pequenas placas en los bloques para mostrar informacion en forma de texto.

# Usage
- `Clic izquierdo` en un bloque para colocar una nueva etiqueta.
- Puedes editar el tamano y el texto de una etiqueta apuntando a ella y presionando la tecla `V`.
