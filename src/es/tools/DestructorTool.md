# Description
El Destructor Tool es una herramienta que, al equiparla, permite al jugador eliminar componentes apuntados.

# Usage
`Manten clic derecho` para activar el modo de destruccion.
`Clic izquierdo` sobre el componente para destruirlo.


---
>- *No puedes destruir cables, bloques o vigas con esta herramienta.*
>- *No puedes eliminar un asiento si esta ocupado por un jugador.*
>- *Si se ha anadido un [OwnerPad](../components/miscellaneous/OwnerPad.md), debes tener el permiso `Build` para destruir un componente.*
