# Description
Der Mini Router ist eine kleinere Variante des [Router](Router.md), die nur über 14 Datenanschlüsse verfügt.

Er benötigt eine Niederspannungs-Stromversorgung zum Betrieb und verbraucht 50 Watt.

Wie der Standard-Router ermöglicht er die Zuweisung von Aliassen an Komponenten und kann mit anderen Routern verkettet werden, um das Netzwerk zu erweitern.

> Der Mini Router, der direkt mit einem Computer kommuniziert, muss mit Strom versorgt werden. Andere Router in der Kette benötigen keinen Strom und können als [Data Bridge](DataBridge.md) verwendet werden, wobei sie weiterhin in der Lage sind, Aliasse und Bildschirmreferenzen aufzulösen.
