# Description
Die Data Bridge ist eine Komponente, die einfach die Verlagerung eines Daten-Endpunkts an eine andere Position ermöglicht.

> Die Data Bridge verlagert den Daten-Endpunkt buchstäblich zur Data Bridge selbst.
>
> Um auf die betreffenden Daten zu verweisen, sollte die Data Bridge per Alias referenziert werden, anstatt die Komponente, von der Sie lesen möchten.
