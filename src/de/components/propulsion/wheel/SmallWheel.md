# Description
Das Small Wheel ist ein einfaches Rad, das nur bremsen und lenken kann und keine Federung hat.
Es benötigt keine Energie zum Betrieb.
# Usage
Das Rad kann zur Steuerung mit einem Computer oder einem anderen Gerät verbunden werden.
