# Description
Der Temperature Sensor ist eine Komponente, die die Temperatur der Umgebung misst, in der er platziert ist.

# Usage
Sobald er auf Ihrem Build platziert ist, kann er zum Beispiel mit einem Computer verbunden werden, um die Umgebungstemperatur abzurufen. Die Temperatur wird in Kelvin angegeben.

### List of outputs
|Channel|Function|Value|
|---|---|---|
|0|Temperature (K)|number|
