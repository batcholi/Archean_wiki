# Description
Der Linear Velocity Sensor ist eine Komponente, die die lineare Geschwindigkeit auf 3 Achsen (X, Y, Z) in Metern pro Sekunde misst.

# Usage
Sobald er auf Ihrem Build platziert ist, kann der Sensor mit einem Computer verbunden werden, um die lineare Geschwindigkeit abzurufen.

### List of outputs
|Channel|Function|value|
|---|---|---|
|0|Linear Velocity X|m/s|
|1|Linear Velocity Y|m/s|
|2|Linear Velocity Z|m/s|
