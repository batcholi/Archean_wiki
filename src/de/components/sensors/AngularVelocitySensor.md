# Description
Der Angular Velocity Sensor ist eine Komponente, die die Winkelgeschwindigkeit auf 3 Achsen (X, Y, Z) in Umdrehungen pro Sekunde misst.

# Usage
Sobald er auf Ihrem Build platziert ist, kann der Sensor mit einem Computer verbunden werden, um die Winkelgeschwindigkeit abzurufen.

### List of outputs
|Channel|Function|Value|
|---|---|---|
|0|Angular velocity X|rot/s|
|1|Angular velocity Y|rot/s|
|2|Angular velocity Z|rot/s|
