# Description
Der Speed Sensor ermöglicht es, die relative Geschwindigkeit zum übergeordneten Objekt (Planet, Mutterschiff...) über seinen Datenport zu senden.

# Usage
Sobald er auf Ihrem Build platziert ist, kann er zum Beispiel mit einem Computer verbunden werden, um Ihre Geschwindigkeit in Metern pro Sekunde abzurufen. Die Ausrichtung des Speed Sensors hat keinen Einfluss auf seine Funktion.
