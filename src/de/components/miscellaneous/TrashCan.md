# Description
Die TrashCan ist eine Komponente, die die sofortige Zerstörung von Objekten ermöglicht, die in ihr Inventar gelegt wurden.

# Usage
`F` zum Öffnen des TrashCan-Inventars. Nach dem Öffnen können Sie Items einlegen, um sie zu zerstören.

>- Die TrashCan kann mit einer Item-Leitung verbunden werden, um automatisch Items aus einer externen Quelle zu zerstören.
