# Description
Die Treppenstufe ist eine Komponente, die als Platte erscheint und das Erstellen von Treppen ermöglicht.

# Usage
Die Treppenstufe kann auf Blöcken platziert werden und ermöglicht es, sich von einem Block zum anderen nach oben oder unten zu bewegen, indem sie ihren Collider nutzt, der eine dreieckige Form hat (siehe Abbildung unten).

![](StepCollider.png)

Dieses System ermöglicht vollständig anpassbare Treppen.
