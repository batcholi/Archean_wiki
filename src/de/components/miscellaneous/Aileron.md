# Description
Das Aileron ist ein Bauteil, das die Aerodynamik eines Builds beeinflusst.

Seine Wirksamkeit hängt eng mit der Dichte des Mediums zusammen, in dem es betrieben wird, sei es in der Atmosphäre (Flugzeug...) oder im Wasser (Boot...).

# Usage
Das Aileron benötigt keine Energie zum Betrieb, nur einen Wert zwischen `-1.0` und `+1.0` über seinen Datenport.

>- Die Ailerons berechnen keine Verdeckungen wie Blöcke es tun. Dies ermöglicht die Optimierung der Flugeffizienz eines Fahrzeugs, indem man sie in größeren Flächen versteckt, zum Beispiel innerhalb der Tragflächen.
