# Description
Die Lamp ist eine Komponente, die einen großen Bereich mit einem maximalen Winkel von 170° beleuchten kann. Sie eignet sich besonders für die Montage an der Decke zur Beleuchtung von Räumen oder Arbeitsbereichen.

# Usage
Die Lamp muss mit Niederspannung versorgt werden und verbraucht bis zu 1000 W, abhängig von der in ihrem Informationsmenü eingestellten Leistung, das über die `V`-Taste zugänglich ist.

Die Farben der Lamp können über das Informationsmenü der Lampe oder über ihren Datenport geändert werden.

### List of inputs
|Channel|Function|Range|
|---|---|---|
|0|Off/On|0 or 1|
|1|Red|0 to 255|
|2|Green|0 to 255|
|3|Blue|0 to 255|
