# Description

Der Power Converter ist ein Gerät, das Hochspannungsenergie in Niederspannungsenergie und umgekehrt umwandelt.

Dieses Gerät ist beispielsweise nützlich, um einen Computer von einer Hochspannungsbatterie zu versorgen, wenn Anschlüsse verfügbar sind, und um das Hinzufügen einer Niederspannungsbatterie zu vermeiden. Wenn Sie jedoch eine Hochspannungskomponente von einer Niederspannungsbatterie versorgen möchten, sind Sie natürlich weiterhin in Bezug auf die Gesamtleistung durch die Ausgangsleistung der Niederspannungsbatterie begrenzt.
