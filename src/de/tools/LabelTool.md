# Description
Das Label Tool ist ein Werkzeug zum Platzieren kleiner Schilder auf Blöcken, um Informationen in Form von Text anzuzeigen.

# Usage
- `Linksklick` auf einen Block, um ein neues Label zu platzieren.
- Du kannst die Größe und den Text eines Labels bearbeiten, indem du es anvisierst und die `V`-Taste drückst.
