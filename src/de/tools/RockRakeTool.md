# Description
Das RockRake Tool ist ein Werkzeug, mit dem du Steine auf dem Boden schneller sammeln kannst als von Hand. Es wird hauptsächlich im Abenteuermodus als Zwischenstufe zwischen Handsammlung und Bohrer verwendet.

# Usage
Das RockRake Tool bietet zwei Hauptvorteile gegenüber dem Sammeln von Hand:
- Das Gedrückthalten der `linken Maustaste` ermöglicht es dir, kontinuierlich Steine aufzusammeln, während die Hand nur einen nach dem anderen aufnehmen kann.
- Die `rechte Maustaste` ermöglicht es dir, die Zusammensetzung des anvisierten Steins zu erfahren.
