# Installation
## Configuration minimale requise
- Carte graphique NVIDIA RTX avec les derniers pilotes
- Microsoft Windows 10 ou 11 ou une distribution Linux moderne

## Achat et jeu
Le jeu est disponible à l'achat sur Steam. Recherchez simplement "Archean" sur le store Steam ou visitez notre page Steam à l'adresse <https://store.steampowered.com/app/2941660/Archean/>.

Après l'achat, le jeu sera disponible dans votre bibliothèque Steam pour le téléchargement.

## Installation
Une fois téléchargé via Steam, lancez le jeu directement depuis votre bibliothèque Steam. Steam gère automatiquement les mises à jour et la gestion des fichiers.

## Cartes graphiques AMD
<font color="orange">
Archean peut fonctionner avec une carte graphique AMD (architecture RDNA2 ou plus récente - série RX 6000 et supérieure). Les performances peuvent être inférieures par rapport aux cartes NVIDIA en raison de deux facteurs :

- La technologie DLSS de NVIDIA offrant une meilleure optimisation
- Un support plus mature du ray-tracing dans les pilotes NVIDIA

Certains glitchs et artefacts peuvent apparaître avec les cartes AMD. Si vous avez besoin d'aide pour optimiser vos paramètres avec une carte AMD, n'hésitez pas à rejoindre notre [communauté Discord](https://discord.gg/GeuskE6CmF) pour obtenir de l'aide et des informations complémentaires.
</font>

## Cartes graphiques Intel ARC
<font color="red">
Les cartes graphiques Intel ARC ne sont pas compatibles avec Archean en raison de l'absence de support des flottants en double précision dans les shaders (shaderfloat64).
</font>
