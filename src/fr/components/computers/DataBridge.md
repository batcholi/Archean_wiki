# Description
Le Data Bridge est un composant qui permet simplement de déplacer un point de terminaison de données vers une autre position.

> Le Data Bridge déplace littéralement le point de terminaison de données vers le Data Bridge lui-même.
>
> Pour référencer ces données, le Data Bridge doit être référencé par alias au lieu du composant depuis lequel vous essayez de lire.
