# Description
La DataJunction permet la transmission de données depuis un seul port vers quatre ports différents, pour diffuser des données vers plusieurs composants. Elle fonctionne de manière unidirectionnelle, vous ne pouvez donc pas lire en retour depuis celle-ci.
