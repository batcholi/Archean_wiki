# Description
La TrashCan est un composant qui permet la destruction instantanee des objets places dans son inventaire.

# Usage
`F` pour ouvrir l'inventaire de la TrashCan. Une fois ouvert, vous pouvez deposer des objets pour les detruire.

>- La TrashCan peut etre connectee avec un conduit d'objets pour detruire automatiquement les objets provenant d'une source externe.
