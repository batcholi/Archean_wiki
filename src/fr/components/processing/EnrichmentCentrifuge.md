# Description
Le composant Enrichment Centrifuge separe un melange de molecules en deux flux distincts en fonction de leur masse moleculaire.

# Usage
Il necessite :
- Une alimentation electrique continue de 1 000 Watts.
- Une entree de donnees pour l'activer.

Le fluide est injecte par un port situe tout en haut du composant. En bas, les molecules les plus legeres sortent par un port positionne plus haut, tandis que les molecules les plus lourdes sortent par un port plus bas situe directement en dessous.

Les deux ports de sortie doivent etre correctement connectes pour assurer un fonctionnement correct.
