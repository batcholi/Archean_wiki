# Description
La Lamp est un composant qui permet d'eclairer une grande zone avec un angle maximum de 170°. Elle est particulierement adaptee pour etre placee au plafond pour eclairer des pieces ou des zones de travail.

# Usage
La Lamp doit etre alimentee en basse tension et consomme jusqu'a 1000 W selon la puissance definie dans son menu d'information accessible via la touche `V`.

Les couleurs de la lampe peuvent etre modifiees via le menu d'information de la lampe ou via son port de donnees.
