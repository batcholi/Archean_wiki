# Description
- Tension : 480 à 500 volts
- Courant : 200 000 ampères par port
- Capacité totale : 120 kWh
- Ports électriques : 4
- Port de données : 1 port pour la surveillance de l'état de la batterie

> **Recyclage :** Lorsqu'elle est broyée, cette batterie restitue environ **50%** de son coût de fabrication en minerais bruts.

# Usage
Avec la batterie fournissant 200 000 ampères par port, vous pouvez obtenir jusqu'à 100 MW sur chaque port pour alimenter des composants.
