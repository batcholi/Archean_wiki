# Description
La Small Wheel est une roue simple qui ne peut que freiner et diriger et ne dispose pas de suspension.
Elle ne necessite aucune energie pour fonctionner.
# Usage
La roue peut etre connectee a un ordinateur ou un autre dispositif pour le controle.
