# Description
Le capteur de vitesse angulaire est un composant qui mesure la vitesse angulaire sur 3 axes (X, Y, Z) en rotations par seconde.

# Usage
Une fois place sur votre construction, le capteur peut etre connecte a un ordinateur pour recuperer la vitesse angulaire.

### Liste des sorties
|Channel|Function|Value|
|---|---|---|
|0|Angular velocity X|rot/s|
|1|Angular velocity Y|rot/s|
|2|Angular velocity Z|rot/s|
