# Description
Le Linear Velocity Sensor est un composant qui mesure la vitesse lineaire sur 3 axes (X, Y, Z) en metres par seconde.

# Usage
Une fois place sur votre construction, le capteur peut etre connecte a un ordinateur pour recuperer la vitesse lineaire.

### Liste des sorties
|Channel|Function|value|
|---|---|---|
|0|Linear Velocity X|m/s|
|1|Linear Velocity Y|m/s|
|2|Linear Velocity Z|m/s|
