# Description
Le capteur de temperature est un composant qui mesure la temperature de l'environnement dans lequel il est place.

# Usage
Une fois place sur votre construction, il peut etre connecte a un ordinateur, par exemple, pour recuperer la temperature de l'environnement. La temperature est fournie en Kelvin.

|Channel|Function|Value|
|---|---|---|
|0|Temperature (K)|number|
