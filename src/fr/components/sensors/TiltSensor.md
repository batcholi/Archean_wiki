# Description
Le capteur d'inclinaison permet d'envoyer via son port de donnees l'inclinaison actuelle par rapport a l'horizon.

# Usage
Une fois place sur votre construction, il peut etre connecte a un ordinateur, par exemple, pour recuperer votre inclinaison entre `-1.0` et `+1.0`.
Son axe long est a utiliser comme un indicateur de niveau.
`-1.0` ou `+1.0` signifie une inclinaison de 90 degres avec une extremite pointant parfaitement vers le bas et l'autre parfaitement vers le haut, tandis que `0.0` signifie a niveau avec l'horizon.

> Ne fonctionne pas dans l'espace
