# Description
Le capteur de vitesse permet d'envoyer la vitesse relative par rapport a l'objet parent (planete, vaisseau mere...) via son port de donnees.

# Usage
Une fois place sur votre construction, il peut etre connecte a un ordinateur, par exemple, pour recuperer votre vitesse en metres par seconde. L'orientation du capteur de vitesse n'a aucun impact sur son fonctionnement.
