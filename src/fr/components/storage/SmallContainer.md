# Description
Le Small Container est un composant de stockage d'une capacite de 25 emplacements.
Il est equipe de deux ports pour connecter des conduits d'objets afin de recevoir ou d'envoyer des objets.

Le port de donnees permet de recuperer le contenu du conteneur sous forme de chaine en utilisant le [systeme key-value](/xenoncode/documentation.md#key-value-objects).

>- *Le Container n'a pas la capacite de tirer ou pousser directement des objets depuis ses ports, c'est uniquement un composant de stockage.*
