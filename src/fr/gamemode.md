# Modes de jeu
Dans Archean, il existe différents modes de jeu. Chaque mode de jeu possède ses propres règles et objectifs selon le type de joueur que vous êtes.

### Creative
En mode Creative, vous avez accès à tous les blocs et objets du jeu directement dans votre inventaire. Vous pouvez construire librement toutes les structures et véhicules que vous souhaitez. Voici les caractéristiques spécifiques du mode Creative :
- Les Blueprint peuvent être chargés sans avoir les composants nécessaires.
- Le système de checkpoint OwnerPad sauvegarde l'état des batteries et des réservoirs.
- Le raccourci de réapparition accessible via `Shift+F10` vous permet de réapparaître sans lâcher le contenu de votre inventaire.
- L'extraction minière et la fabrication sont disponibles, mais pas nécessaires pour construire des structures ou des véhicules.
- La température des fluides peut être réglée manuellement via GetInfo (V) sur les réservoirs et les volumes.

> Vous pouvez réinitialiser votre inventaire en utilisant le bouton RESET disponible en haut de votre inventaire.


### Adventure
En mode Adventure, vous commencez avec un inventaire vide et devez collecter des ressources pour fabriquer les composants nécessaires à la construction de structures et de véhicules. Voici les caractéristiques spécifiques du mode Adventure :
- Les Blueprint ne peuvent pas être chargés sans avoir les composants nécessaires dans votre inventaire.
- Le système de checkpoint OwnerPad ne restaure pas l'état des batteries et des réservoirs.
- Le raccourci de réapparition accessible via `Shift+F10` dépose le contenu de votre inventaire dans une boîte à votre emplacement avant la réapparition.
- L'extraction minière et la fabrication sont nécessaires pour construire des structures ou des véhicules.
- **L'inventaire de l'avatar est limité à 45 emplacements.**
- **Les objets dans votre inventaire ont une masse**, ce qui affecte votre vitesse de déplacement et la masse de toute construction à bord de laquelle vous vous trouvez.

### Survival
Le mode Survival n'est pas encore disponible dans le jeu. Il sera ajouté dans une future mise à jour.

## Changer le mode de jeu d'un monde existant
Vous pouvez changer le mode de jeu d'un monde existant à tout moment depuis le launcher.

1. Dans le menu jouer en solo, cliquez sur l'**icône d'engrenage** (⚙️) sur le côté droit de l'entrée du monde
2. Sélectionnez **Edit**
3. Modifiez le paramètre du mode de jeu
4. Sauvegardez vos modifications

Vous pouvez également renommer ou dupliquer des mondes depuis ce menu.
