# Description
Le Destructor Tool est un outil qui, une fois équipé, permet au joueur de supprimer les composants ciblés.

# Usage
`Maintenez le clic droit` pour activer le mode destruction.
`Clic gauche` sur le composant pour le détruire.


---
>- *Vous ne pouvez pas détruire les câbles, les blocs ou les poutres avec cet outil.*
>- *Vous ne pouvez pas supprimer un siège s'il est occupé par un joueur.*
>- *Si un [OwnerPad](../components/miscellaneous/OwnerPad.md) a été ajouté, vous devez avoir la permission `Build` pour détruire un composant.*
