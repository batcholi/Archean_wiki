# Description
Le RockRake Tool est un outil qui vous permet de collecter les roches au sol plus rapidement qu'à la main. Il est principalement utilisé en mode aventure comme niveau intermédiaire entre la collecte manuelle et la foreuse minière.

# Usage
Le RockRake Tool offre deux avantages principaux par rapport à la collecte manuelle :
- Maintenir le `bouton gauche de la souris` vous permet de ramasser les roches en continu, alors que la main ne peut en ramasser qu'une à la fois.
- Le `bouton droit de la souris` vous permet de connaître la composition de la roche ciblée.
