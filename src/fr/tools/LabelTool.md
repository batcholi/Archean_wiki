# Description
Le Label Tool est un outil permettant de placer de petites plaques sur les blocs pour afficher des informations sous forme de texte.

# Usage
- `Clic gauche` sur un bloc pour placer un nouveau label.
- Vous pouvez modifier la taille et le texte d'un label en le visant et en appuyant sur la touche `V`.
