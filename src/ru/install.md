# Installation
## Минимальные системные требования
- Видеокарта NVIDIA RTX с последними драйверами
- Microsoft Windows 10 или 11, либо современный дистрибутив Linux

## Покупка и запуск
Игра доступна для покупки в Steam. Просто найдите "Archean" в магазине Steam или посетите нашу страницу Steam по адресу <https://store.steampowered.com/app/2941660/Archean/>.

После покупки игра будет доступна в вашей библиотеке Steam для загрузки.

## Installation
После загрузки через Steam запустите игру прямо из вашей библиотеки Steam. Steam автоматически управляет обновлениями и файлами.

## Видеокарты AMD
<font color="orange">
Archean может работать с видеокартами AMD (архитектура RDNA2 или новее -- серия RX 6000 и выше). Производительность может быть ниже по сравнению с картами NVIDIA по двум причинам:

- Технология DLSS от NVIDIA обеспечивает лучшую оптимизацию
- Более зрелая поддержка драйверов трассировки лучей на картах NVIDIA

На картах AMD могут появляться графические артефакты. Если вам нужна помощь с оптимизацией настроек для карты AMD, присоединяйтесь к нашему [Discord-сообществу](https://discord.gg/GeuskE6CmF) для получения помощи и дополнительной информации.
</font>

## Видеокарты Intel ARC
<font color="red">
Видеокарты Intel ARC несовместимы с Archean из-за отсутствия поддержки чисел с плавающей запятой двойной точности в шейдерах (shaderfloat64).
</font>
