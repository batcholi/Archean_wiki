# Mining

Добыча ресурсов в Archean -- это процесс поиска, сбора и переработки минеральных ресурсов, находящихся в рельефе небесных тел. Это основа системы крафта -- без руды вы не сможете ничего построить.

Полный цикл добычи: **Сканирование → Сбор → Дробление → Крафт**.

## Поиск руд

Прежде чем собирать камни, следует определить, где сконцентрированы ценные руды.

[Ore Scanner Tool](tools/OreScannerTool.md) -- это ручной прибор, отображающий в реальном времени график плотности руды в направлении, куда вы смотрите. Просто возьмите его и осмотритесь -- это самый быстрый способ разведать местность.

Для автоматического сканирования компонент [Ore Scanner](components/mining/OreScanner.md) можно управлять через [XenonCode](xenoncode/documentation.md), и он возвращает данные о составе в виде [текстовых объектов ключ-значение](xenoncode/documentation.md#key-value-objects).

---

## Сбор камней

Камни -- это небольшие объекты, разбросанные по поверхности небесных тел. Они различаются по размеру, и **чем больше камень, тем больше материала** вы получите при сборе.

### Вручную
Без экипированного инструмента нажмите `ЛКМ` для подбора камня. Для каждого камня нужно нажимать отдельно.

### Rock Rake Tool
[Rock Rake](tools/RockRakeTool.md) -- рекомендуемый способ ручного сбора камней:

- **Удерживайте `ЛКМ`**: непрерывный подбор камней при наведении
- **`ПКМ`**: осмотр камня перед подбором -- показывает его состав и выход

---

## Автоматическая добыча -- Mining Drill

[Mining Drill](components/mining/MiningDrill.md) автоматически извлекает камни из рельефа. Его необходимо установить на закреплённой конструкции с помощью [Ground Anchor](components/miscellaneous/GroundAnchor.md).

Эффективность бура снижается с глубиной -- следует **периодически перемещать его**, когда выход падает. Подробные характеристики см. на странице [Mining Drill](components/mining/MiningDrill.md).

## Состав камней

Каждый камень имеет **состав** -- набор элементов с соответствующими плотностями, определяемый его расположением на местности. Состав представляется в виде [текстовых объектов ключ-значение](xenoncode/documentation.md#key-value-objects):

`.Al{0.1500}.Fe{0.2000}.Si{0.6500}` -- что означает 15% алюминия, 20% железа, 65% кремния.

Камни с **одинаковым составом** складываются в один слот инвентаря. Камни с разным составом занимают отдельные слоты.

### Элементы

|Символ|Элемент|Символ|Элемент|
|---|---|---|---|
|Al|Алюминий|Ni|Никель|
|C|Углерод|Si|Кремний|
|Cr|Хром|Ag|Серебро|
|Cu|Медь|Sn|Олово|
|Au|Золото|Ti|Титан|
|F|Флюорит|W|Вольфрам|
|Fe|Железо|U|Уран|
|Pb|Свинец|Ch|???|

## Переработка камней

Чтобы превратить камни в пригодные для использования руды, их необходимо раздробить. Есть два способа:

### Обработка аватаром (начало игры)
В вашем рюкзаке есть встроенная **Item Processing Area** (1 слот), доступная из [интерфейса крафта](interfaces.md#item-processing-area). Поместите камни в этот слот и нажмите **Crush**, чтобы превратить их в руду прямо из инвентаря. Это медленно, но не требует инфраструктуры -- именно так вы получаете первые руды в режиме Adventure.

### Компонент Crusher (автоматизация)
[Crusher](components/mining/Crusher.md) -- это размещаемый компонент для крупномасштабной переработки. Он преобразует камни в руды пропорционально их составу. Например, дробление 10 кг камня с 20% железа даёт 2000 г железной руды.

При очень низких концентрациях Crusher накапливает дробные количества по нескольким камням, пока не сможет произвести хотя бы 1 единицу -- ничего не теряется.

> Crusher **не вытягивает** из своего входа -- камни должны быть [переданы](pushpull.md) ему. Подробнее см. на странице [Push & Pull](pushpull.md).

## Распределение ресурсов

Распределение руд зависит от небесного тела, высоты и близости к вулканическим образованиям. Подробнее см. [Earth](celestialbodies/earth.md) и [Moon](celestialbodies/moon.md).

|Редкость|Элементы|Где искать|
|---|---|---|
|Обычные|Si, Fe, C, Cu, Sn|Повсеместно / выше уровня моря|
|Необычные|Al, Ag|Высокогорье|
|Редкие|Ni, Cr, Ti, W|Высокогорье / очень большая высота|
|Очень редкие|Au, Pb, U, F|Глубоководье / вблизи вулканов|

- **Вулканы** увеличивают содержание железа и свинца и являются **единственным источником флюорита**

## Практические советы

- **Сначала разведайте**: используйте Ore Scanner Tool, чтобы найти место, богатое нужными вам рудами, прежде чем размещать инфраструктуру
- **Типичные схемы**: Mining Drill → Crusher → Container или Mining Drill → Container → [Conveyor](components/items/ItemConveyor.md) → Crusher
- **Размеры**: 1 Crusher обрабатывает до 4 Mining Drill при высоком напряжении
- **Перемещайте буры**, когда эффективность значительно падает
- **Rock Rake для начала игры**: используйте его для сбора первого железа и крафта пути к Mining Drill
- **Проверяйте состав**: нажмите ПКМ по камню с Rock Rake, чтобы узнать его содержимое перед сбором
