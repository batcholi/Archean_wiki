# Description
Label Tool — это инструмент для размещения небольших табличек на блоках для отображения информации в виде текста.

# Usage
- `Левая кнопка мыши` на блоке для размещения новой метки.
- Вы можете изменить размер и текст метки, нацелившись на неё и нажав клавишу `V`.
