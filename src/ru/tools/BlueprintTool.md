# Description
Blueprint Tool позволяет сохранить копию постройки. Blueprint сохраняются в папке `(Client Settings)/blueprints` в формате `.json`, по одному файлу на каждый Blueprint с соответствующим именем.

С помощью этой системы вы можете переносить свои постройки в другие миры или на серверы, делиться ими с друзьями или просто дублировать постройку.

# Usage

## Scanning a Build
1. **Левая кнопка мыши** для входа в режим сканирования (дальность: 100 метров)
2. Наведитесь на постройку, которую хотите сохранить
3. **Удерживайте левую кнопку мыши** одну секунду для сканирования и сохранения Blueprint
4. Введите имя во всплывающем диалоге

### Updating an Existing Blueprint
Чтобы обновить существующий Blueprint, просто введите точно такое же имя, как у существующего Blueprint. Кнопка **Create** изменится на **Update**, позволяя перезаписать предыдущую версию с сохранением идентификатора Steam Workshop, если он был опубликован.

## Spawning a Blueprint
1. **Правая кнопка мыши** для открытия интерфейса Blueprint
2. Используйте **строку поиска** для фильтрации Blueprint по имени
3. Выберите Blueprint из вкладки **Local** или **Steam Workshop**
4. Нажмите **Spawn** для входа в режим размещения

### Placement Controls

#### Free Placement (new build)
| Действие | Управление |
|--------|---------|
| Подтвердить размещение | **Удерживайте левую кнопку мыши** (2 секунды) |
| Вращение | **Колесо мыши** |
| Привязка к земле и установка как статичный | **Удерживайте Shift** |
| Отмена | **Правая кнопка мыши** |

#### On an Existing Build
При наведении на существующую постройку **быстро нажмите левую кнопку мыши**, чтобы прикрепить Blueprint и войти в **режим Gizmo**:

| Действие | Управление |
|--------|---------|
| Перемещение / Вращение | **Ручки Gizmo** (перетаскивание) |
| Переключение обзора мышью | **Правая кнопка мыши** |
| Подтвердить размещение | **Удерживайте левую кнопку мыши** вне Gizmo (2 секунды), или кнопка **Spawn** |
| Отмена | **Правая кнопка мыши** (когда не в режиме обзора мышью) |

### Spawning in Space
Находясь в космосе рядом с существующей постройкой, вы можете размещать Blueprint так же, как при [создании новой постройки](ConstructorTool.md#creating-a-build-in-space).

## Steam Workshop
- Нажмите **Publish** рядом с локальным Blueprint, чтобы загрузить его в Steam Workshop
- При публикации доступен встроенный инструмент для создания скриншотов

# Notes
- В режиме приключений отображаются необходимые предметы с их статусом (зелёный = доступен, красный = отсутствует, оранжевый = мод не активен)
- Если был добавлен [OwnerPad](../components/miscellaneous/OwnerPad.md), у вас должно быть разрешение "Build" для сохранения Blueprint
