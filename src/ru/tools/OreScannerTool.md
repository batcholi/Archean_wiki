# Description
Ore Scanner Tool — это ручное устройство, отображающее минеральный состав в направлении, куда вы смотрите. Он показывает график плотности руды в зависимости от расстояния в реальном времени, позволяя находить месторождения руды перед добычей.

Этот инструмент отличается от [компонента OreScanner](../components/mining/OreScanner.md) — в то время как компонент предназначен для установки на транспортные средства и управления через XenonCode, Ore Scanner Tool предоставляет непосредственный визуальный интерфейс для игрока.

# Usage

## Basic Operation
1. Экипируйте Ore Scanner Tool из инвентаря
2. Направьте в сторону, которую хотите просканировать
3. Инструмент отображает график, показывающий концентрации руды в этом направлении

## Interface
Окно сканера показывает:
- **Distance selector**: Выбор дальности сканирования (100 м, 250 м, 500 м или 1000 м)
- **Ore checkboxes**: Выбор руд для отображения на графике
- **Graph**: Показывает плотность руды (ось Y) в зависимости от расстояния (ось X)

## Scanning Tips
- Направление сканирования основано на горизонтальной составляющей направления вашего взгляда
- График обновляется автоматически при перемещении или изменении направления
- Выберите несколько руд для сравнения их расположения
- Iron (Fe) выбран по умолчанию как наиболее распространённый ресурс

---

> **Примечание:** Ore Scanner Tool требует нахождения на небесном теле с данными о рельефе. Он не будет работать в космосе или в средах без данных о составе местности.
