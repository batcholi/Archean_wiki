# Description
Paint Tool позволяет красить блоки, кабели, метки и компоненты пользовательскими цветами и типами поверхности.

# Usage

## Controls
| Действие | Управление |
|--------|---------|
| Покрасить грань | **Левая кнопка мыши** |
| Покрасить весь блок (все грани) | **Shift** + **Левая кнопка мыши** |
| Выбрать цвет с блока | **Правая кнопка мыши** |
| Заменить цвет по всей постройке | **X** + **Левая кнопка мыши** |

## Color Palettes
Paint Tool использует систему палитр для сохранения и организации ваших цветов.

### Palette Management
- **Create**: Создать новую пустую палитру
- **Copy**: Дублировать текущую палитру с новым именем
- **Delete**: Удалить текущую палитру

### Adding Colors
Нажмите кнопку **+** в палитре, чтобы добавить новую ячейку цвета.

## Color Picker
Нижняя часть интерфейса содержит:

1. **RGB Color Picker**: Выберите любой цвет с помощью полосы оттенка и квадрата насыщенности/значения
2. **Gamma Preview**: Показывает, как цвет будет выглядеть в игре с вашими настройками гаммы

## Surface Materials
Каждый цвет может иметь свой тип поверхности:

| Материал | Внешний вид |
|----------|------------|
| **Matte** | Шероховатая, неотражающая поверхность |
| **Glossy** | Гладкая, блестящая поверхность |
| **Metal** | Шероховатая металлическая поверхность |
| **Chrome** | Зеркальная металлическая поверхность |
| **Transparent** | Прозрачная (для эффектов стекла) |

## Replace Color
Удерживайте **X** и нажмите **левую кнопку мыши** на любой покрашенной поверхности, чтобы заменить этот цвет по всей постройке. Это работает с:
- **Блоки**: Заменяет все блоки с тем же индексом цвета
- **Трубы/Кабели**: Заменяет все трубы с тем же цветом
- **Компоненты**: Заменяет все компоненты того же типа и материала

## Symmetry Painting
Когда у постройки включён [режим симметрии](ConstructorTool.md), Paint Tool автоматически красит зеркальный блок. Это применяется к:
- Покраске отдельной грани
- Покраске всего блока (**Shift**)

Если блок находится на плоскости симметрии, вместо этого красится зеркальная грань на том же блоке.

> **Примечание**: Замена цвета (**X**) всегда применяется ко всей постройке независимо от симметрии, поэтому симметрия в этом режиме не применяется.

# Notes
- Покраска блока применяет цвет **на грань** (используйте Shift для всех граней)
- Кабели имеют дополнительные параметры настройки, см. [Spool](../consumables/Spool.md#painting-cables)
