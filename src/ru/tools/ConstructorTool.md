# Description
Constructor Tool --- основной инструмент для строительства в Archean. Он позволяет создавать новые постройки, добавлять или удалять рамы, а также размещать блоки различных форм.

# Build Modes
Нажмите **C**, чтобы открыть радиальное меню и выбрать режим строительства:

|Режим|Описание|
|---|---|
|**Frame**|Создание и изменение структурной рамы (стальные балки)|
|**Cube**|Размещение кубических блоков|
|**Slope**|Размещение наклонных блоков|
|**Corner**|Размещение угловых блоков|
|**Pyramid**|Размещение блоков-пирамид|
|**InvCorner**|Размещение инвертированных угловых блоков|
|**Triangle**|Размещение треугольных панелей|
|**Wall / Platform**|Размещение плоских стеновых или напольных панелей|
|**Strut**|Размещение тонких структурных стальных балок|

# Materials
При размещении блоков (не рам) используйте **колесо мыши** с зажатой клавишей **C** для выбора материала.

## Material Masses

| Материал | Масса за единицу |
|----------|---------------|
| Composite | 0,25 кг |
| Aluminium | 0,5 кг |
| Steel | 1 кг |
| Glass | 1 кг |
| Concrete | 10 кг |
| Lead | 150 кг |

> **Примечание:** Стальные балки (рамы) весят **10 кг** каждая.

### How Block Mass is Calculated
Базовой единицей является куб **25×25×25 см**. Общая масса блока зависит от:

1. **Размер**: Более крупные блоки содержат больше единиц (например, блок 50×50×50 см = 8 единиц)
2. **Форма**: Некубические формы (наклоны, углы, пирамиды, инвертированные углы) весят **50%** от кубического эквивалента, так как занимают меньший объём
3. **Материал**: Каждый материал имеет свою массу за единицу (см. таблицу выше)

**Формула:** `Масса = единицы × коэффициент_формы × масса_материала`
- `единицы` = (размер_x) × (размер_y) × (размер_z) с шагом 25 см
- `коэффициент_формы` = 1.0 для кубов и стен, 0.5 для наклонов, углов, пирамид и инвертированных углов. Треугольники имеют переменную стоимость, зависящую от их площади.

### Why Masses Don't Match Reality
Массы намеренно упрощены:
- Значения **округлены** для удобства чтения
- Блоки не являются на 100% сплошными — они представляют структурные панели с внутренним каркасом, а не монолитные куски материала

# Removing Elements
Чтобы удалить любой элемент (блок, треугольник, балку или стену), наведитесь на него, удерживая **правую кнопку мыши**, затем быстро нажмите **левую кнопку мыши**. Это работает из любого режима строительства.

> Удаление рам имеет собственное управление (см. режим Frame ниже).

# Usage

## Frame Mode

### Creating a New Build
Удерживайте **левую кнопку мыши** одну секунду, затем отпустите. Будет создана новая постройка с одной металлической рамой 3x3x3.

> **Совет:** Удерживайте **Shift** при создании, чтобы выровнять новую раму по земле и сделать её статичной (закреплённой).

### Creating a Build in Space
Находясь в космосе рядом с существующей постройкой, вы можете создать новую постройку, которая появится **в 5 метрах перед вами**. Новая постройка создаётся как дочерняя относительно ближайшей корневой постройки. Это также работает с [Blueprint Tool](BlueprintTool.md).

### Adding Frames
Наведитесь на грань существующей рамы и быстро нажмите **левую кнопку мыши**.

### Removing Frames
Наведитесь на грань существующей рамы, удерживая **правую кнопку мыши**, затем быстро нажмите **левую кнопку мыши**.

### Adding Individual Beams
Наведитесь на место в существующей раме, где должна быть балка, удерживая **Shift**, затем быстро нажмите **левую кнопку мыши**.

### Removing Individual Beams
Наведитесь на балку существующей рамы, удерживая **Shift** и **правую кнопку мыши**, затем быстро нажмите **левую кнопку мыши**.

## Block Modes (Cube, Slope, Corner, Pyramid, InvCorner)

### Adding Blocks
1. Наведитесь на блок или балку
2. Нажмите **левую кнопку мыши** для размещения блока
3. Используйте **колесо мыши** для вращения (кроме кубов)
4. Удерживайте **левую кнопку мыши** и перетаскивайте для изменения размера
5. Используйте **колесо мыши**, удерживая **левую кнопку мыши**, для изменения размера по другому измерению

> **Совет профи:** Плоскость перетаскивания/масштабирования определяется нормалью грани блока, на которую вы наводитесь. Колесо мыши масштабирует вдоль этой нормали, а перетаскивание масштабирует по двум другим осям.

> **Совет:** Удерживайте **Shift** перед нажатием **левой кнопки мыши**, чтобы скопировать блок, на который вы наводитесь, или логически добавить следующий блок.

## Triangle Mode

Режим Triangle позволяет размещать треугольные панели, привязанные к сетке постройки. Эти панели полезны для создания изогнутых форм, аэродинамических поверхностей или любой непрямоугольной геометрии.

### Placing Triangles
1. Нажмите на **3 точки сетки** (вершины существующих блоков, треугольников или балок), чтобы создать треугольник
2. Или нажмите на **существующее ребро**, чтобы начать с 2 заранее выбранных вершин — тогда потребуется только один дополнительный клик
3. Используйте **колесо мыши** при размещении, чтобы **перевернуть направление нормали** (управляет тем, какая сторона треугольника обращена наружу)
4. Удерживайте **Shift** на осевой грани блока для **точности под-сетки**, привязки к полной сетке 25 см
5. Нажмите **правую кнопку мыши**, чтобы отменить последнюю размещённую вершину

> Треугольники привязываются к вершинам блоков, других треугольников и конечным точкам балок. Максимальный размах — 4 м по каждой оси.

### Materials
Удерживайте **C** и используйте **колесо мыши** для выбора материала. Треугольники поддерживают все те же материалы, что и блоки.

### Smooth Shading
Нажмите **X**, наведясь на треугольник, чтобы переключить плавное затенение на всех связанных треугольниках в одной группе. Плавное затенение применяется с использованием алгоритма заливки: оно распространяется через **общие рёбра** на все соседние треугольники, образующие непрерывную поверхность.

Для корректной работы плавного затенения:
- Вершины треугольников должны **точно совпадать** на сетке — соседние треугольники должны иметь одинаковые позиции вершин
- Отдельные группы треугольников (не соединённые общими рёбрами) обрабатываются как **независимые группы сглаживания** — переключение сглаживания в одной группе не влияет на другую

> Плавное затенение также учитывает режим симметрии.

## Strut Mode

Балки — это тонкие структурные стальные элементы, соединяющие две точки сетки. Они полезны для создания ферм, антенн, лесов или лёгких структурных элементов.

### Placing Struts
1. Нажмите на **2 точки сетки**, чтобы создать балку между ними
2. Балки привязываются к вершинам блоков, треугольников и других балок
3. Нажмите **правую кнопку мыши**, чтобы отменить первую размещённую вершину

> Максимальный размах — 4 м по каждой оси. В режиме приключений для балок требуются предметы **Steel Rod**.

### Subdivision Snapping
Удерживайте **Shift**, наведясь на существующую балку, чтобы привязаться к точкам подразделения вдоль неё. Используйте **колесо мыши**, удерживая **Shift**, чтобы изменить шаг подразделения:

|Размеры шага|
|---|
|1 м|
|50 см|
|25 см (по умолчанию)|
|10 см|
|5 см|

Это позволяет размещать новые балки или треугольники с точными интервалами вдоль существующей балки.

## Wall / Platform Mode

### Adding Walls
Наведитесь на грань рамы и быстро нажмите **левую кнопку мыши**.

> **Совет:** Удерживайте **Shift**, чтобы использовать другую логическую грань рамы, на которую вы наводитесь.

> **Совет:** Удерживайте **Shift** при размещении, чтобы привязаться к земле и создать статичную платформу прямо перед вами.

# Symmetry Mode
Constructor Tool поддерживает симметричное строительство, позволяющее автоматически создавать зеркальные структуры.

Симметрию можно включить через меню **GetInfo** постройки (нажмите **V** на любой раме). При включении:
- Все операции с рамами и блоками зеркально отражаются относительно оси симметрии
- Позиция оси симметрии может быть настроена с **точностью 0,125 м**, что позволяет размещать зеркало по центру блоков

# Adventure Mode
В режиме приключений вам нужны необходимые материалы в инвентаре:
- **Steel Beams** для рам
- **Блочные предметы** (Composite, Concrete, Steel и т.д.) для блоков

Инструмент покажет, сколько материалов требуется для каждой операции.
