# Description
RockRake Tool — это инструмент, позволяющий собирать камни с земли быстрее, чем вручную. Он в основном используется в режиме приключений как промежуточный уровень между ручным сбором и буровой установкой.

# Usage
RockRake Tool предлагает два основных преимущества по сравнению с ручным сбором:
- Удержание `левой кнопки мыши` позволяет непрерывно подбирать камни, тогда как вручную можно подбирать только по одному.
- `Правая кнопка мыши` позволяет узнать состав целевого камня.
