# Description
Spool Tool используется для размещения кабелей и труб, соединяющих компоненты между собой, позволяя передавать данные, энергию, предметы или жидкости между ними.

# Spool Types
Нажмите **C**, чтобы открыть меню выбора катушки. Доступно 5 типов катушек:

<p align="center">
  <img src="../consumables/dataspool.png" />
  <img src="../consumables/lowvoltagespool.png" />
  <img src="../consumables/highvoltagespool.png" />
  <img src="../consumables/fluidspool.png" />
  <img src="../consumables/itemconduitspool.png" />
</p>

|Тип|Цвет|Назначение|
|---|---|---|
|**Data Cable**|Синий|Соединение портов данных для передачи информации|
|**Low Voltage Cable**|Красный|Соединение портов низковольтного питания|
|**High Voltage Cable**|Оранжевый|Соединение портов высоковольтного питания|
|**Fluid Pipe**|Серый|Соединение портов жидкостей для передачи жидкостей/газов|
|**Item Conduit**|Тёмно-серый|Соединение портов предметов для передачи предметов|

Каждый тип катушки может подключать только совместимые порты. Катушки можно складывать в инвентаре, а оставшаяся длина отображается на каждом предмете-катушке.

# Usage

## Selecting Spool Type
Нажмите **C**, чтобы открыть меню выбора катушки и выбрать тип кабеля, который вы хотите проложить.

## Creating a Cable (Connecting Two Components)
1. Наведитесь на разъём компонента и нажмите **левую кнопку мыши**, чтобы начать кабель
2. Нажимайте, чтобы добавить промежуточные точки для формирования пути кабеля
3. Наведитесь на целевой разъём и нажмите **левую кнопку мыши**, чтобы завершить соединение

Во время создания кабеля:
- **Правая кнопка мыши** удаляет последнюю размещённую точку (или отменяет, если точек нет)
- **Колесо мыши** переключает между альтернативными маршрутами автопрокладки
- Удерживайте **Shift** для привязки кабелей к поверхностям компонентов
- Удерживайте **X** для размещения кабеля на внутренней стороне блоков/компонентов

## Auto Path-Finding
Spool Tool оснащён автоматической прокладкой маршрутов, предлагающей пути для кабелей. Используйте **колесо мыши** при размещении, чтобы переключаться между различными вариантами маршрутов.

## Creating a Flexible Cable
Чтобы соединить компоненты на разных постройках:
1. Начните кабель на одной постройке
2. Завершите его на компоненте другой постройки

Это создаёт **гибкий кабель**, который:
- Физически связывает две постройки
- Ограничивается физическим движком
- Не имеет ограничения по силе (не отсоединяется)
- Подвержен воздействию гравитации

Вы также можете создать гибкий кабель между двумя компонентами **одной постройки**, удерживая **X**.

## Deleting a Cable
Удерживайте **правую кнопку мыши**, затем быстро нажмите **левую кнопку мыши** на существующем кабеле, чтобы удалить его.

## Painting Cables
Используйте [Paint Tool](PaintTool.md) для настройки внешнего вида кабелей:
- Обычная покраска меняет цвет кабеля
- Удерживайте **Shift** для эффекта полос
- Удерживайте **X** для замены цвета на всех совпадающих кабелях
- Комбинируйте оба для прозрачных полос

---

> **Советы:**
> - Если кабель не создаётся, возможно, у вас недостаточно оставшейся длины в текущей катушке
> - Кабели не имеют ограничений передачи или потерь, связанных с длиной
> - Кабели не определяют направление передачи
> - Кабель нельзя изменить после размещения — его нужно удалить и создать заново
> - Гибкие кабели влияют на производительность больше, чем обычные — старайтесь использовать обычные кабели, где это возможно
> - Инструменты могут использовать предметы из внешних контейнеров, если поместить инструмент в этот контейнер
