# Description
Dashboard Tool используется для проектирования приборных панелей с более мелкими компонентами в более гибком формате. Он позволяет размещать экраны, кнопки, светодиоды, метки и другие элементы на поверхностях для создания пользовательских панелей управления и дисплеев.

# Usage
Нажмите `C`, чтобы открыть радиальное меню и выбрать элемент, который вы хотите разместить.

Нажмите `V`, чтобы открыть меню GetInfo элемента и получить доступ к дополнительным параметрам.

## Available Elements

### Screens

| Тип | Разрешение | Максимальный размер |
|------|------------|----------|
| Standard Screen | 2 пикселя/см (200 px/м) | 4 м |
| HD Screen | 6 пикселей/см (600 px/м) | 50 см |

HD Screen обеспечивает в 3 раза большую плотность пикселей для более детальных дисплеев, но ограничен размером 50 см.

Экраны обновляются каждые 20 мс.

**Параметры GetInfo (`V`):**
- **Matte**: Переключение между глянцевой и матовой поверхностью

### Labels
Labels функционируют как экраны, но с разрешением 5 пикселей/см (500 px/м) и максимальным размером 1 м.

Labels обновляются каждые 500 мс, что делает их более подходящими для статичных или медленно меняющихся текстовых дисплеев.

**Параметры GetInfo (`V`):**
- **Text**: Введите текст для отображения (поддерживается многострочный ввод)
- **Text Align**: Center, Left, Right, Top, Bottom, Top Left, Bottom Left, Top Right, Bottom Right
- **Text Size**: от 1 до 8
- **Color picker**: Установите цвет текста

### Buttons

#### [PushButton](../components/controllers/PushButton.md)
Отправляет сигнал при нажатии.

**Параметры GetInfo (`V`):**
- **Single Pulse**: При включении отправляет только один импульс за нажатие вместо непрерывного сигнала при удержании

#### [ToggleButton](../components/controllers/ToggleButton.md)
Переключается между состояниями вкл/выкл при нажатии.

**Параметры GetInfo (`V`):**
- **Allow IO input**: Принимать входные сигналы для управления состоянием
- **Horizontal**: Переключение горизонтальной ориентации

#### ArrowButton
Направленная кнопка с вариантами вращения.

**Параметры GetInfo (`V`):**
- **Rotation**: 0-3 (шаг 90°)
- **Single Pulse**: Аналогично PushButton

### LED
LED можно нажимать как кнопки, и они могут отображать пользовательские цвета.

**Параметры GetInfo (`V`):**
- **Single Pulse (press)**: Отправлять только один импульс за нажатие
- **Color From Input**: Получать значение цвета из канала данных вместо использования настроенного цвета
- **Color picker**: Установите цвет LED

### Trim
Trim — это декоративный элемент, который можно размещать с точностью до 1 см и изменять размер с шагом 1 см до 4 м. Полезен для добавления визуального разделения или рамок к макету приборной панели.

## Adding Components to Dashboards
Любой компонент можно разместить на приборной панели в произвольных позициях и ориентациях. Это позволяет встраивать приборы, датчики или другие компоненты непосредственно в дизайн вашей приборной панели.
