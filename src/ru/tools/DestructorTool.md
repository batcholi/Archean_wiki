# Description
Destructor Tool — это инструмент, который при экипировке позволяет игроку удалять целевые компоненты.

# Usage
`Удерживайте правую кнопку мыши`, чтобы активировать режим разрушения.
`Нажмите левую кнопку мыши` на компоненте, чтобы уничтожить его.


---
>- *Вы не можете уничтожить кабели, блоки или балки этим инструментом.*
>- *Вы не можете удалить сиденье, если его занимает игрок.*
>- *Если был добавлен [OwnerPad](../components/miscellaneous/OwnerPad.md), у вас должно быть разрешение `Build` для уничтожения компонента.*
