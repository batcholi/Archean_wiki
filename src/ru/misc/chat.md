# Chat & Commands

## Система чата
Чат позволяет игрокам общаться друг с другом. Чтобы написать сообщение другим игрокам, нажмите `Enter`, чтобы открыть чат, затем введите сообщение и снова нажмите `Enter` для отправки.
> - Вы можете скопировать сообщение, просто кликнув по нему `левой кнопкой мыши`. Многострочные сообщения полностью поддерживаются.
> - Максимальная длина сообщения ограничена 800 символами.
> - Переписка чата записывается в файл журнала сервера.

### История команд
Когда чат открыт, используйте **стрелки вверх/вниз** для просмотра ранее введённых команд. Над полем ввода появляется визуальное всплывающее окно, показывающее до 20 последних записей. Вы также можете кликнуть на любую запись, чтобы выбрать её.

> **Примечание:** Если видны подсказки команд (автодополнение), стрелки переключают подсказки. Просмотр истории возобновляется, когда подсказки не отображаются.

## Команды
Команды позволяют выполнять определённые действия, вводя их в чат с префиксом `/`. Доступно несколько команд, часть из которых зарезервирована для администраторов.

#### Список команд, доступных всем игрокам:
- `/clear` -- Очищает сообщения, отображаемые в чате.
- `/help` -- Отображает список доступных команд и их описания.
- `/list` -- Отображает список подключённых игроков.
- `/info` -- Отображает информацию о сервере. *(Имя сервера, нагрузка CPU, тики в секунду (25 тиков/с -- идеальное значение для стабильного сервера))*

#### Список команд, доступных администраторам:
- `/save` -- Сохраняет текущее состояние игры.
- `/setspawn <username>` -- Устанавливает точку возрождения для конкретного пользователя.
- `/give [<stack>] [<moduleName>.]<itemType> [<inventoryAlias|username>]` -- Выдаёт предметы игроку. Если имя пользователя не указано, предметы выдаются вам.
- `/kick <username|id>` -- Выгоняет игрока с сервера.
- `/ban <username|id>` -- Банит игрока на сервере.
- `/timeadd [-]<hours>` -- Прибавляет или вычитает время в игре.
