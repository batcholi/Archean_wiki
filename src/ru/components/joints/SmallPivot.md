# Description
Small Pivot — компонент, включающий строительный вращающийся блок. Предназначен для обеспечения вращения объектов на постройке.

> <font color="green"> *Этот компонент связан с герметизацией построек. Подробнее см. страницу [Pressurization](../../pressurization.md).*</font>

# Usage
Small Pivot может работать в двух режимах: Servo (по умолчанию) и Velocity. Для переключения между режимами нажмите клавишу V, чтобы открыть информационный интерфейс компонента.

В этом интерфейсе доступны две дополнительные настройки:
- `Max Rotation Speed` — определяет максимальную скорость вращения в оборотах в секунду.
- `Acceleration` — определяет ускорение (в оборотах в секунду за секунду), с которым поворотный механизм будет разгоняться до Max Rotation Speed.

## Servo Mode
В режиме servo устройство поворачивается в точную позицию, определяемую данными, полученными через порт данных. Принимает нормализованные значения от `-1.0` до `+1.0`, что соответствует повороту от `-360°` до `+360°`. Это означает, что значения `0.0`, `-1.0` и `+1.0` приведут к одной и той же позиции servo.

## Velocity Mode
В режиме velocity устройство непрерывно вращается в направлении, указанном данными из порта, принимая значения от `-1.0` до `+1.0` для определения скорости и направления вращения. `1.0` означает Max Rotation Speed.

> - Постройки, установленные на подвижной части, не могут сталкиваться с родительской или соседней постройкой. Они могут сталкиваться только с рельефом или другими отдельными постройками.
> - Чтобы уничтожить Small Pivot, необходимо сначала удалить все дочерние блоки/компоненты, которые он содержит.
