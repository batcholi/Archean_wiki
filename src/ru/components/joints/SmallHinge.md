# Description
Small Hinge — компонент, включающий строительный блок на петле.

> <font color="green"> *Этот компонент связан с герметизацией построек. Подробнее см. страницу [Pressurization](../../pressurization.md).*</font>

# Usage
Small Hinge работает только в режиме servo.

При нажатии V для открытия информационного интерфейса доступны две настройки:
- `Max Rotation Speed` — определяет максимальную скорость вращения в оборотах в секунду.
- `Acceleration` — определяет ускорение (в оборотах в секунду за секунду), с которым петля будет разгоняться до Max Rotation Speed перед замедлением для достижения целевой позиции.

Устройство поворачивается в точную позицию, определяемую данными, полученными через порт данных. Принимает нормализованные значения от `-1.0` до `+1.0`, что соответствует повороту от `-90°` до `+90°`.

> - Постройки, установленные на подвижной части, не могут сталкиваться с родительской или соседней постройкой. Они могут сталкиваться только с рельефом или другими отдельными постройками.
> - Чтобы уничтожить Small Hinge, необходимо сначала удалить все дочерние блоки/компоненты, которые он содержит.
### List of outputs
|Channel|Function|Value|
|---|---|---|
|0|Angle|-1.0 to +1.0|
|1|Speed|rot/s|
