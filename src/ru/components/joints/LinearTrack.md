# Description
Linear Track — компонент, включающий строительный подвижный блок. Предназначен для обеспечения линейного перемещения объектов на постройке. Перемещение возможно только вдоль направляющей, автоматически создаваемой по оси движения блока.

> <font color="green"> *Этот компонент связан с герметизацией построек. Подробнее см. страницу [Pressurization](../../pressurization.md).*</font>

# Usage
Linear Track может работать в двух режимах: Servo (по умолчанию) и Velocity. Для переключения между режимами нажмите клавишу V, чтобы открыть информационный интерфейс компонента.

В этом интерфейсе доступны дополнительные настройки:
- `Max Speed` — определяет максимальную скорость в метрах в секунду.
- `Acceleration` — определяет ускорение, с которым линейная направляющая будет разгоняться до Max Speed.
- `Override limits` — при включении позволяет вручную задать минимальные и максимальные пределы позиции вместо автоматически рассчитанных значений на основе доступного пространства направляющей.

## Направляющие
Направляющие для компонента Linear Track автоматически создаются и обновляются вдоль его оси. Они создаются поверх обычных блоков любого типа. Для завершения направляющей линию необходимо прервать, например, поставив другой блок сверху или оставив отверстие вдоль направляющей.

## Servo Mode
В режиме servo устройство перемещается в точную позицию, определяемую данными, полученными через порт данных. Принимает все значения и реагирует в диапазоне между min и max. Если получено число больше или меньше пределов, устройство переместится в соответствующую позицию min/max.

## Velocity Mode
В режиме velocity устройство непрерывно движется в направлении, указанном данными из порта, принимая значения от `-1.0` до `+1.0` для определения скорости и направления движения. `1.0` означает Max Speed.

> - Постройки, установленные на подвижной части, не могут сталкиваться с родительской или соседней постройкой. Они могут сталкиваться только с рельефом или другими отдельными постройками.
> - Чтобы уничтожить Linear Track, необходимо сначала удалить все дочерние блоки/компоненты, которые он содержит.
> - Linear Track имеет максимальный диапазон перемещения от -250 м до +250 м (всего 500 м).
