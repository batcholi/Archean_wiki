# Description
Propeller — компонент, создающий тягу с помощью вращающихся лопастей. Используется для движения транспортных средств в воздухе или воде.

# Usage

Настраивается через интерфейс конфигурации, доступный по клавише `V`.

### Интерфейс конфигурации
Предоставляет информацию о пропеллере и позволяет настраивать его.
#### Информация
- `Input Voltage`: входное напряжение в вольтах.
- `Power`: потребляемая мощность в ваттах.
- `Thrust`: создаваемая тяга в ньютонах.
- `Rotation Speed`: скорость вращения в оборотах в секунду.
- `Pitch`: нормализованный угол шага.

#### Конфигурация
- `Radius`: радиус пропеллера в метрах.
- `Twist`: угол закрутки пропеллера (нормализованный).
- `Blades`: количество лопастей.

### Энергия
Пропеллер имеет порт низковольтного и высоковольтного питания, что позволяет в большей или меньшей степени контролировать подаваемую мощность.
### Низковольтное питание
В этой конфигурации пропеллер потребляет до 50 кВт.
#### Высоковольтное питание
В этой конфигурации пропеллер потребляет до 500 кВт.
