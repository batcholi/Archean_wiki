# Description
Система реактивного управления (RCS) состоит из газовых двигателей малой тяги, используемых преимущественно для управления ориентацией и положением космического аппарата. Также применяется для точной корректировки при стыковке. RCS состоит из нескольких малых двигателей, которые можно быстро включать/выключать по отдельности для точного управления.

# Usage
RCS может работать на различных жидкостях, которые влияют на его характеристики в зависимости от плотности и давления. Управляется компьютером или другим устройством для регулировки тяги и направления.

Не выполняет никакого сгорания.
