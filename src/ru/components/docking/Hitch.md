# Description  
Компонент Hitch соединяет две конструкции в фиксированной точке, позволяя при этом полную свободу вращения вокруг сочленения. После соединения транспортные средства остаются зафиксированными по положению, но свободно вращаются друг относительно друга вокруг сцепки.

# Usage  

Hitch имеет **порт данных**, управляющий его состоянием.  
Когда два компонента Hitch активированы (armed) и оказываются на близком расстоянии, они автоматически соединяются.
Соединение можно разорвать в любой момент, деактивировав любой из компонентов.

> - По соображениям производительности столкновения между сцеплёнными конструкциями не обрабатываются. В игре используется серверная физика, и постоянная проверка столкновений между транспортными средствами на близком расстоянии была бы слишком затратной. В будущем обновлении может появиться возможность задавать ограничения по всем осям сцепки для моделирования физических пределов, например, чтобы прицеп не поворачивался на слишком большой угол и не проходил сквозь тягач.
