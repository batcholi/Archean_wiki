# Description
SlopeLight — компактный осветительный компонент, предназначенный для установки на наклонных поверхностях. Излучает свет с углом луча по умолчанию 120°.

# Usage
SlopeLight должен быть подключён к низковольтному питанию и потребляет до 1000 Вт в зависимости от мощности, установленной в информационном меню, доступном по клавише `V`.

Цвет и угол освещения можно настроить через информационное меню (клавиша `V`) или через порт данных.

### List of inputs
|Channel|Function|Range|
|---|---|---|
|0|Off/On|0 or 1|
|1|Red|0 to 255|
|2|Green|0 to 255|
|3|Blue|0 to 255|

### Настройка через клавишу V
- **Max Power**: регулируется от 0 до 1000 Вт (по умолчанию: 200 Вт)
- **Angle**: регулируется от 20° до 120°
- **RGB**: палитра для выбора цвета освещения
