# Description
Lamp — компонент, позволяющий освещать большую область с максимальным углом 170°. Особенно подходит для размещения на потолке для освещения комнат или рабочих зон.

# Usage
Lamp должна быть подключена к низковольтному питанию и потребляет до 1000 Вт в зависимости от мощности, установленной в информационном меню, доступном по клавише `V`.

Цвет лампы можно изменить через информационное меню или через порт данных.

### List of inputs
|Channel|Function|Range|
|---|---|---|
|0|Off/On|0 or 1|
|1|Red|0 to 255|
|2|Green|0 to 255|
|3|Blue|0 to 255|
