# Description
OwnerPad — универсальное устройство безопасности, обеспечивающее защиту построек от потенциальных вредоносных действий.

Помимо функции безопасности, он позволяет сохранять состояние постройки или транспортного средства (позиция, скорость, уровень заряда батарей, переключатели...).  
С помощью этой функции можно мгновенно вернуться к сохранённому состоянию одним нажатием.

# Usage
### Кнопки SAVE и RESET
Кнопка `SAVE` сохраняет позицию постройки и состояние её компонентов, а кнопка `RESET` возвращает постройку к состоянию на момент последнего сохранения.

### Кнопка ROLES
Роли позволяют разрешать/запрещать действия на постройке, как показано на примере. Они предназначены для обеспечения безопасности ваших построек от вредоносных действий других игроков.
При размещении OwnerPad на постройке, если другой OwnerPad отсутствует, вы становитесь владельцем этой постройки.

По умолчанию все разрешения установлены на `@Everyone`, что делает постройку полностью публичной.  
Помимо `@Everyone` и `dksm (OWNER)` в примере, отображается список всех подключённых игроков, что позволяет назначать роли выбранным игрокам.

Рядом с каждым подключённым игроком есть кнопка `Give Ownership` для передачи владения выбранному игроку. После передачи у вас не будет прав на постройку, пока новый владелец не назначит вам роли.

### Управление через порт данных
OwnerPad имеет порт данных для управления с компьютера или удалённо, например, через Beacon.
Необходимо отправить текст в канал 0, содержащий ключевое слово `"save"` или `"reset"` в зависимости от желаемого действия.

Необходимо отправить пустой текст `""` для сброса состояния перед отправкой следующей команды.
