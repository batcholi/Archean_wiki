# Description
TrashCan — компонент, позволяющий мгновенно уничтожать объекты, помещённые в его инвентарь.

# Usage
Нажмите `F`, чтобы открыть инвентарь TrashCan. После открытия вы можете поместить предметы для уничтожения.

>- TrashCan можно подключить предметной линией для автоматического уничтожения предметов, поступающих из внешнего источника.
