# Description
Aileron — устройство, влияющее на аэродинамику постройки.

Его эффективность тесно связана с плотностью среды, в которой он работает, будь то атмосфера (самолёт...) или вода (лодка...).

# Usage
Aileron не требует энергии для работы — только значение от `-1.0` до `+1.0` через порт данных.

>- Элероны не рассчитывают затенение, в отличие от блоков. Это позволяет оптимизировать лётную эффективность транспортного средства, скрывая их внутри более крупных поверхностей, например, внутри крыльев.
