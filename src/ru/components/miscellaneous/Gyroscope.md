# Description
Gyroscope — компонент, который при подаче питания и активации гасит угловую скорость. В основном используется для стабилизации транспортного средства или остановки углового момента в невесомости.

# Power Supply
SmallGyroscope питается от **низкого напряжения**, а Gyroscope — от **высокого напряжения**. Они потребляют больше энергии при запуске, затем постепенно снижают потребление по мере достижения скорости вращения, заданной через порт данных.

# Usage
Для запуска гироскоп должен получить значение от `0.0` до `1.0` в порт данных для уменьшения/увеличения скорости вращения, тем самым усиливая стабилизирующий эффект.

Гироскоп позволяет вручную ориентировать транспортное средство через порт данных, используя индуцированный крутящий момент инерционных колёс внутри него. Он действует в зависимости от своей ориентации и скорости вращения.

### List of inputs
|Channel|Function|range|
|---|---|---|
|0|Speed| 0.0 to 1.0|
|1|Control| -1.0 to +1.0|

>Эффект гироскопа ограничен относительно массы постройки. Для усиления стабилизирующего эффекта можно увеличить количество гироскопов.
