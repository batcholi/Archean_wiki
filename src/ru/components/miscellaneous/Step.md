# Description
Step — компонент в виде пластины, позволяющий создавать лестницы.

# Usage
Ступень может быть размещена на блоках и обеспечивает подъём или спуск с одного блока на другой благодаря своему коллайдеру треугольной формы (см. рисунок ниже).

![](StepCollider.png)

Эта система позволяет создавать полностью настраиваемые лестницы.
