# Description
Speed Sensor позволяет передавать через порт данных относительную скорость по отношению к родительскому объекту (планета, материнский корабль и т.д.).

# Usage
После установки на постройку его можно подключить, например, к компьютеру для получения скорости в метрах в секунду. Ориентация датчика скорости не влияет на его работу.
