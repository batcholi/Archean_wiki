# Description
Temperature Sensor — компонент, измеряющий температуру окружающей среды, в которой он находится.

# Usage
После установки на постройку его можно подключить, например, к компьютеру для получения температуры окружающей среды. Температура указывается в Кельвинах.

### List of outputs
|Channel|Function|Value|
|---|---|---|
|0|Temperature (K)|number|
