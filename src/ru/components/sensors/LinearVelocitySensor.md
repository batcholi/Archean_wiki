# Description
Linear Velocity Sensor — компонент, измеряющий линейную скорость по 3 осям (X, Y, Z) в метрах в секунду.

# Usage
После установки на постройку датчик можно подключить к компьютеру для получения линейной скорости.

### List of outputs
|Channel|Function|value|
|---|---|---|
|0|Linear Velocity X|m/s|
|1|Linear Velocity Y|m/s|
|2|Linear Velocity Z|m/s|
