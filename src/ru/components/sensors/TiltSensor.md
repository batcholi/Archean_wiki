# Description
Tilt Sensor позволяет передавать через порт данных текущий наклон относительно горизонта.

# Usage
После установки на постройку его можно подключить, например, к компьютеру для получения наклона в диапазоне от `-1.0` до `+1.0`.  
Его длинная ось используется как уровень.  
`-1.0` или `+1.0` означает наклон на 90 градусов, когда один конец направлен строго вниз, а другой — строго вверх, а `0.0` означает горизонтальное положение.

> Не работает в космосе
