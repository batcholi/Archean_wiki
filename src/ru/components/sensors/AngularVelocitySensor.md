# Description
Angular Velocity Sensor — компонент, измеряющий угловую скорость по 3 осям (X, Y, Z) в оборотах в секунду.

# Usage
После установки на постройку датчик можно подключить к компьютеру для получения угловой скорости.

### List of outputs
|Channel|Function|Value|
|---|---|---|
|0|Angular velocity X|rot/s|
|1|Angular velocity Y|rot/s|
|2|Angular velocity Z|rot/s|
