# Description
PlayerSensor обнаруживает игроков в пределах настраиваемой 3D-области относительно позиции и ориентации датчика. Выводит информацию об обнаруженных игроках в виде текстового объекта key-value.

# Usage
После установки на постройку PlayerSensor обнаруживает любого игрока, входящего в зону обнаружения. Зона обнаружения представляет собой 3D-параллелепипед, настраиваемый через меню клавиши `V`.

Датчик выводит информацию об игроках на канале 0 в виде текстового объекта key-value, содержащего:
- ID игрока
- Имя пользователя
- Расстояние от датчика

### Настройка через клавишу V
- **box_min**: минимальные координаты области обнаружения (X, Y, Z) — по умолчанию: -10, -10, -10
- **box_max**: максимальные координаты области обнаружения (X, Y, Z) — по умолчанию: +10, +10, +10

Координаты области обнаружения задаются относительно позиции и ориентации датчика.

### List of outputs
|Channel|Function|Value|
|---|---|---|
|0|Detected players|key-value text|

### Формат вывода
Вывод представляет собой текстовый объект key-value: `.p<playerID>{.username{<name>}.distance{<meters>}}`
