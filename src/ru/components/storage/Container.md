# Description
Container — компонент хранения с ёмкостью 50 слотов.
Оснащён двумя портами для подключения предметных линий для приёма или отправки предметов.

Порт данных позволяет получать содержимое контейнера в виде строки с помощью системы [key-value](/xenoncode/documentation.md#key-value-objects).

>- *Контейнер не имеет возможности самостоятельно забирать или отправлять предметы через свои порты — это исключительно компонент хранения.*
