# Description
Pilot Seat позволяет игроку управлять (отправлять значения по различным каналам) компонентом, используя привязанные элементы управления транспортным средством с клавиатуры, контроллера или джойстика.

# Usage
Нажмите `R`, чтобы сесть в кресло.
Нажмите `R`, чтобы выйти из кресла.

> Вы можете включить **Hold key to Exit Seat** в настройках игры, если предпочитаете прежнее поведение с удержанием R в течение одной секунды.

> Сидя в кресле, можно пересесть в другое ближайшее кресло, не выходя из текущего, с помощью клавиши `R`.
> При выходе из кресла запоминается ваше положение относительно конструкции в момент посадки, и именно туда вы вернётесь при выходе.

## Configuration
В информационном окне Pilot Seat, доступном по клавише `V`:

| Option | Description |
|--------|-------------|
| **Smooth Controls** | Сглаживает ввод с клавиатуры для игроков без джойстика |
| **Emit user token on Channel 0** | Отправляет токен пользователя на канал 0 вместо `1` (включено по умолчанию) |

Токен пользователя можно использовать с [HUD Controller](../miscellaneous/HudController.md) для определения, какой пользователь сидит в кресле.
