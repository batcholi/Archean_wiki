# Description
Toggle Button постоянно отправляет значение при активации или деактивации. По умолчанию он отправляет `1` во включённом состоянии и `0` в выключенном, но можно настроить пользовательские значения.

# Usage
Клавиша `F` используется для переключения Toggle Button.

## Configuration
В меню настроек, доступном по клавише `V`:

| Option | Description |
|--------|-------------|
| **Dual-Sided** | Делает кнопку доступной с обеих сторон |
| **Allow IO Input** | Позволяет управлять кнопкой с другого компонента через порт данных |
| **Off Value** | Значение, отправляемое при выключенной кнопке (по умолчанию: `0`) |
| **On Value** | Значение, отправляемое при включённой кнопке (по умолчанию: `1`) |
