# Description
Numpad — это компонент, предоставляющий сенсорную цифровую клавиатуру для отправки числовых значений другим компонентам.

# Usage
Вы можете ввести числовое значение с помощью сенсорных кнопок цифровой клавиатуры, нажав клавишу `F`. Введённые цифры отображаются на экране Numpad, но вступают в силу/обновляются только при нажатии кнопки подтверждения (зелёной).

Жёлтая кнопка позволяет удалить последнюю введённую цифру, а красная — очистить всё.

> - Если текущее значение отрицательное, его можно сделать положительным, нажав сенсорную кнопку `-`.
> - При нажатии кнопки подтверждения на канал 1 отправляется значение `1` на 1 тик, в остальное время отправляется `0`.
