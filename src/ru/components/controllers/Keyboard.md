# Description
Keyboard — это компонент, предоставляющий сенсорную клавиатуру для отправки буквенно-цифровых значений другим компонентам.

# Usage
Вы можете ввести буквенно-цифровое значение с помощью сенсорных кнопок клавиатуры, нажав клавишу `F`. Введённые символы отображаются на экране Keyboard, но вступают в силу/обновляются только при нажатии кнопки подтверждения (зелёной).

Жёлтая кнопка позволяет удалить последний введённый символ, а красная — очистить всё.

> - `^` Позволяет переключаться между строчными и прописными буквами
> - `!?` Отображает специальные символы
> - При нажатии кнопки подтверждения на канал 1 отправляется значение `1` на 1 тик, в остальное время отправляется `0`.
