# Description
Компонент Volume при добавлении на постройку автоматически вычисляет и определяет занимаемый объём. Его можно использовать для:
- Создания топливных баков произвольной формы.  
- Обеспечения герметизации кабин для определённых сред (подводные лодки, ракеты и т.д.).  
- Создания дирижаблей...
- ...  

> <font color="green"><strong> Этот компонент связан с герметизацией построек. Подробнее см. страницу [Pressurization](../../pressurization.md).</strong></font>

# Usage
Компонент Volume прост в использовании и не требует специальной настройки. Просто добавьте его на постройку и нажмите кнопку `Scan`, чтобы автоматически определить объём герметичного отсека, в котором он расположен. Обратите внимание, что он вычисляет только объём того отсека, в котором находится, а не всей постройки. Для каждого отсека, который нужно герметизировать, необходимо установить отдельный компонент Volume.

Этот компонент может отображать два состояния:
- `Airtight`: состояние становится зелёным, если отсек полностью герметичен.
- `Leak`: состояние становится красным, если отсек не герметичен.

На экране Volume отображается только объём герметичного отсека в кубических метрах (м³). Для получения более подробной информации откройте информационное окно, нажав клавишу `V`. Доступные данные включают:
- `Volume capacity (m³)`: общая ёмкость герметичного объёма.
- `Contents Mass (kg)`: общая масса содержимого в объёме.
- `Pressure (kPa)`: давление внутри герметичного отсека.
- `Liquid Level (%)`: процент заполнения жидкостью.
- `Composition`: распределение различных жидкостей в виде нормализованных процентов.

В режиме Creative появляются дополнительные кнопки для заполнения или опустошения объёма.
