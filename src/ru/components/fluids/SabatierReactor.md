# Description
Sabatier Reactor — компонент, позволяющий преобразовывать углекислый газ (CO2) и водород (H2) в метан (CH4).

# Usage
Sabatier Reactor требует высоковольтного питания и потребляет до 10 кВт во время работы. Он захватывает CO2 непосредственно из окружающего воздуха, поэтому должен находиться в среде, содержащей CO2.

Для запуска процесса преобразования просто подключите источник водорода к жёлтому входному порту. Произведённый метан можно собирать с красного выходного порта.

### Производство
Sabatier Reactor может обрабатывать до 0,1 кг водорода в секунду, производя 0,2 кг метана в секунду.

> Sabatier Reactor выделяет воду (H2O) в окружающую среду в ходе реакции. Обычно эта вода не видна, но в герметичном объёме она начнёт накапливаться.
