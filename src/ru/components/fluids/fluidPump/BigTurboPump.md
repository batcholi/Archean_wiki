# Description
Big Turbo Pump — компонент, используемый для перекачки жидкости высокой плотности со скоростью до 100 кг в секунду.

# Usage
## Питание
Для работы насоса требуется высоковольтное питание. Он потребляет до 100 кВт при работе на полной скорости.

## Данные
Порт данных позволяет управлять скоростью насоса, отправляя значение от `-1` до `1`.
При подключении насоса к компьютеру можно также получать текущий расход.

> При отправке отрицательного значения насос будет работать в обратном направлении.
