# Description
Item Conveyor — компонент, позволяющий перемещать предметы из одной точки в другую. Он работает, забирая предметы на входе и отправляя их на выходе. Может использоваться, например, для транспортировки предметов из одного контейнера в другой.

# Usage
Item Conveyor требует низковольтного питания, потребление энергии прямо пропорционально скорости перемещения предметов.

Его можно настроить через информационное окно (доступное по клавише `V`) или через порт данных.
Доступные параметры настройки:

|Option|Value|Description|
|---|---|---|
|**Pull n items/sec**|1 to 1000|Максимальное количество предметов для перемещения в секунду|
|**Pull x kg/sec**|0 to 1000 kg|Максимальная масса предметов для перемещения в секунду|
|**Filter**|text|Позволяет задать белый список для определённого типа предметов. Например, если указано "Wire", будут пропускаться только провода|

> - Item Conveyor может фильтровать только один тип предметов за раз. Для перемещения нескольких типов предметов необходимо использовать несколько конвейеров.
> - Фильтр чувствителен к регистру.
