# Description
Crusher — компонент, позволяющий быстро дробить породу для получения руды.

Также может использоваться для **переработки любого крафтового предмета** обратно в исходные материалы. Процесс переработки возвращает **100% основных ресурсов**, использованных для создания предмета, рекурсивно разбирая рецепт крафта.

> **Примечание:** [Батареи](../energy/battery/LowVoltageBattery.md) имеют специальный рецепт переработки, возвращающий лишь приблизительно **50%** стоимости крафта. Это предотвращает эксплуатацию системы переработки для получения бесплатных зарядок путём дробления и повторного крафта разряженных батарей.

# Usage
Crusher требует высоковольтного питания и потребляет 10 кВт.

Для использования Crusher просто отправьте породу для дробления через его входной порт предметов. Он не забирает предметы со входа самостоятельно, но способен отправлять полученную руду в любой контейнер, принимающий предметы.

Crusher способен обрабатывать 400 кг породы в секунду, что соответствует производительности 4 [Mining Drill](MiningDrill.md), работающих на полной мощности.

> Когда состав породы содержит низкую концентрацию определённых руд, Crusher будет накапливать руду в малых количествах до тех пор, пока не станет возможным произвести хотя бы одну единицу руды.
