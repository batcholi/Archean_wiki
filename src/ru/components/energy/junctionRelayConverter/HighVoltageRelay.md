# Description
High Voltage Relay — это устройство, которое подаёт питание на компонент, пропуская ток только при получении ненулевого сигнала на порт данных.

> Сторона с двумя портами предназначена для подключения источника питания и порта данных.
>
> Выход энергии расположен на стороне с одним портом.
