# Description
Steam Turbine — это компонент, преобразующий тепловую энергию горячего пара в электрическую энергию.

# Usage
Работает путём подачи горячего водяного пара на вход жидкости. Чем горячее пар и чем выше расход, тем больше энергии можно произвести.

На полной мощности она может выдавать приблизительно **27 мегаватт**.

Необходимо подключение выхода жидкости. Без него турбина не сможет отводить пар и не будет производить энергию.

- Минимальная рабочая температура: 373 K  
- Максимальная эффективная температура: 650 K (оптимальная работа)  
- Максимальный расход: 100 кг/с  

Генерируемая энергия подаётся напрямую на высоковольтный электрический выход.  

Когда мощность не потребляется полностью, турбина автоматически перепускает пар для поддержания максимальной скорости вращения.
Этот эффект приводит к тому, что выходная жидкость не охлаждается так сильно.
