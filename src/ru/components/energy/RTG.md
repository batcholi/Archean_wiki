# Description
Радиоизотопный термоэлектрический генератор (RTG) вырабатывает низковольтную энергию. Он может обеспечивать непрерывную мощность в зависимости от охлаждающей способности окружающей среды.

# Usage
Подключите RTG к компоненту, которому требуется низковольтная энергия.

RTG имеет два электрических порта, что позволяет подключать два компонента одновременно или последовательно соединять несколько RTG для увеличения общей выходной мощности.

### List of outputs
|Channel|Function|
|---|---|
|0|Generated Power (Watts)|
|1|Output Power (Watts)|

> - RTG может генерировать до **12 кВт** мощности. Фактическая выходная мощность зависит от охлаждающей способности окружающей среды (снижается до ~6 кВт в вакууме космоса). Он не оказывает вредного воздействия на игрока или окружающую среду.
>
> - Если вы используете RTG для питания двух компонентов, общая мощность, распределённая между двумя портами, не может превышать доступную выходную мощность RTG.
>
> - Если один из двух компонентов стремится потребить всю доступную мощность RTG, это может лишить другой компонент питания. В этом случае лучше использовать разветвители питания (power junctions), чтобы все компоненты получали энергию равномерно.


# How to produce Plutonium

## Plutonium Production Process

|Step|Inputs|Outputs|Temperature|
|---|---|---|---|
|Crusher|Uranium Ore: 1000 g|Uranium Powder (U) : 1000 g (U235 : 10%, U238 : 90%)|-|
|ChemicalFurnace (Yellow Cake - U₃O₈)|Uranium Powder (U) : 0.714 g, O₂ : 0.128 g|Yellow Cake (U₃O₈) : 0.842 g|750K - 950K|
|ChemicalFurnace (Uranium Dioxide - UO₂)|Yellow Cake (U₃O₈) : 0.842 g, H₂ : 0.004 g|Uranium Dioxide (UO₂) : 0.810 g, H₂O : 0.036 g|850K - 1050K|
|Crafter|Uranium Dioxide (UO₂) : 1000 g|Uranium Rod (U235 : 10%, U238 : 90%) : 1|-|
|Crusher|Uranium Rod|Plutonium Dioxide (PuO₂) : 1 g (Pu : 100%)|-|
|Crafter|Plutonium Dioxide (PuO₂) : 5000 g|Plutonium Pellet : 1|-|

> Рекомендуется использовать низкообогащённый уран (LEU) для производства плутония. Степень обогащения не влияет на количество получаемого плутония.

# Additional Information
<font color="orange">
В реальности переработка плутония из отработанного ядерного топлива — это чрезвычайно сложная и требовательная промышленная операция, требующая передовой инфраструктуры и специализированных объектов. По этой причине извлечение и переработка плутония в настоящее время не поддерживаются в Archean.  

Вместо этого RTG используют упрощённую форму плутония, специально произведённого для генерации энергии. Хотя эта альтернатива остаётся проще в обращении, чем реальный плутоний, её производство намеренно сделано более сложным, чем в предыдущих версиях игры, что делает RTG менее тривиальными для получения, но при этом доступными для продвинутых игроков.
</font>
