# Description

Computer — это компонент, предназначенный для запуска программ XenonCode с целью управления другими компонентами или отображения различной информации на встроенном экране.

Он обладает характеристиками, определяющими его вычислительную мощность, объём хранилища и памяти.
Эта информация отображается в BIOS, как показано на изображении ниже.

![Bios](bios.png)

### Информация BIOS:
- **Processor Type**: Процессор XPU64-AIO обозначает данный тип компонента — компьютер «всё в одном» (All-In-One).
- **Frequency**: Частота — это скорость, с которой компьютер выполняет код текущей программы. Она соответствует серверному параметру `updates_per_second` (по умолчанию 25 тиков в секунду).
- **Max IPC**: Этот параметр задаётся в конфигурационном файле и определяет максимальное количество инструкций за цикл, прежде чем компьютер «виртуально аварийно завершит работу».
- **System RAM**: В программах XenonCode можно хранить значения в изменчивых (volatile) переменных, которые сбрасываются при перезагрузке компьютера или перезагрузке программы. Этот параметр определяет максимальное количество значений для всех переменных текущей программы.
- **Storage Capacity**: Аналогично системной RAM, существует тип переменных для хранения данных на жёстком диске, которые сохраняются постоянно до тех пор, пока не будут изменены. Ёмкость ограничена максимум 256 значениями.
- **I/O Ports**: Этот параметр определяется подключёнными компонентами и просто показывает количество портов данных на компьютере.

> BIOS — это программа, которая запускается по умолчанию на каждом компьютере при загрузке.
>
> В данном случае её роль — отображать характеристики компьютера, а также список доступных программ на жёстком диске, чтобы можно было выбрать одну из них для выполнения.
>
> Можно настроить BIOS, создав новый файл с именем "main.xc", чтобы изменить его внешний вид или автоматически загружать программу. Подробнее см. в документации XenonCode IDE.

# Usage
### Программа:
Когда программа создана и сохранена, она появится в списке программ в BIOS. Выбрать программу для выполнения можно клавишей `F`.
### Кнопки:
На компьютере есть две физические кнопки: кнопка "Code", которая открывает XenonCode IDE для разработки программ, и кнопка "Reboot", которая перезагружает компьютер и повторно запускает программу main.xc (BIOS).
### Питание:
Для работы компьютер требует низковольтного питания. В режиме ожидания он потребляет 30 ватт, а потребление может возрасти до 500 ватт в зависимости от соотношения количества выполняемых инструкций текущей программы к настроенному на сервере значению MAX IPC.
### HDD:
Computer имеет отсек для жёсткого диска. В отсеке можно хранить несколько HDD (активен только один), обмениваться HDD с другими компьютерами или хранить их в инвентаре. Это удобно, когда нужно переместить компьютер, не теряя код.
