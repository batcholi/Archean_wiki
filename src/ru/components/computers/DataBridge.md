# Description
Data Bridge — это компонент, который просто позволяет перенести конечную точку данных (Data) в другое положение.

> Data Bridge буквально перемещает конечную точку данных к самому мосту данных.
>
> Для обращения к этим данным в псевдониме (Alias) следует указывать именно Data Bridge, а не компонент, данные которого вы пытаетесь считать.
