# Interfaces & Inventory Management

На этой странице описаны все системы пользовательского интерфейса в Archean, включая управление инвентарём, горячие клавиши и работу с окнами.

---

## Инвентари аватара

У игрока есть доступ к двум личным инвентарям:

### Пояс

**Пояс** -- это панель быстрого доступа в нижней части экрана.

- Содержит **9 слотов** для инструментов и предметов
- Выбирайте предметы клавишами **1**--**9**
- Собранные камни сначала попадают на пояс, а затем в рюкзак при переполнении
- Активный предмет выделяется зелёной рамкой

### Рюкзак

**Рюкзак** -- это ваше основное хранилище инвентаря.

- Открыть/закрыть клавишей **Tab**
- В режиме Adventure: **48 слотов**
- В режиме Creative: неограниченное количество слотов с предзаполненными предметами
- Есть **строка поиска** для фильтрации предметов по названию
- Масса предметов влияет на скорость перемещения в режиме Adventure

## Система руки

**Рука** -- это ваш текущий выбранный/активный предмет.

| Действие | Описание |
|--------|-------------|
| **Цифровые клавиши (1-9)** | Выбрать слот пояса как активный |
| **Двойной клик** на любой предмет | Выбрать его как активный (работает из любого открытого инвентаря) |

Когда инструмент в руке, вы можете использовать его с помощью:
- **ЛКМ** (основная функция)
- **ПКМ** (вторичная функция)
- **Удерживайте C** для настройки инструмента
- **Удерживайте X** для специального режима

## Взаимодействие с компонентами

Нажмите **F** для взаимодействия с компонентом или экраном, на который вы смотрите (открытие инвентарей, нажатие кнопок, переключение тумблеров, регулировка рычагов, клики по экранам...).

Кроме того, вы можете **кликнуть ЛКМ** по компонентам напрямую:

| Условие | Поведение |
|-----------|----------|
| **Свободная мышь** (без обзора мышью) | Клик по любому видимому компоненту для взаимодействия |
| **Обзор мышью без активного предмета** | Клик для взаимодействия с компонентом в перекрестии |
| **Обзор мышью с активным предметом** | ЛКМ используется инструментом -- взаимодействие с компонентами отключено |

> **Примечание:** Взаимодействие через ЛКМ позволяет нажимать кнопки или переключать тумблеры без необходимости убирать инструмент -- просто освободите мышь или опустошите руку.

## Манипуляции с предметами

### Перетаскивание

| Действие | Описание |
|--------|-------------|
| **Перетащить предмет** | Переместить предмет между слотами или инвентарями |
| **Бросить за пределы инвентаря** | Бросить предмет на землю (создаётся ящик) |

### Стакание и разделение

При перетаскивании стопки предметов:

| Действие | Описание |
|--------|-------------|
| **ПКМ** | Положить **1** предмет из стопки |
| **Shift + ПКМ** | Положить **половину** стопки |
| **Бросить на предмет того же типа** | Сложить предметы вместе (если есть место) |

### Быстрое перемещение

| Действие | Описание |
|--------|-------------|
| **Shift + ЛКМ** | Быстро переместить предмет в другой открытый инвентарь |

Это работает между рюкзаком и любым открытым контейнером. Предметы сначала попытаются сложиться с существующими предметами того же типа.

## Крафт

Откройте рюкзак (**Tab**), чтобы получить доступ к интерфейсу крафта в левой части.

### Действия крафта

| Режим | Доступные кнопки |
|------|-------------------|
| **По умолчанию** | **+1** и **+10** |
| **Удерживая Shift** | **+50** и **+max** (на основе доступных ресурсов) |

Нажмите на ингредиент, чтобы развернуть и увидеть подрецепты (рекурсивный крафт).

### Требования для крафта

- Твёрдые ингредиенты должны быть в рюкзаке (или в любом открытом инвентаре)
- Жидкости (H₂O, H₂, O₂) должны быть предоставлены через **канистры** в специальных слотах для жидкостей внизу окна крафта
- Можно крафтить, используя ресурсы из любого открытого инвентаря -- ингредиенты автоматически перемещаются в рюкзак

### Item Processing Area

Item Processing Area (1 слот) используется для **обработки предметов**, а не для крафта. Например:
- Дробление камней в руду (начало игры в режиме Adventure)

### Слоты для жидкостей

Внизу окна крафта вы можете видеть уровни жидкостей: **Water**, **Hydrogen** и **Oxygen**.

- У вашего аватара есть **встроенный бак для воды**, который автоматически наполняется, когда ваши ноги касаются воды (океан, озеро)
- Переключатель **Electrolysis** преобразует воду в водород и кислород для рецептов крафта, которые их требуют

## Внешние инвентари (контейнеры)

Взаимодействуйте с [Container](components/storage/Container.md) клавишей **F**, чтобы открыть его инвентарь.

### Использование инструментов из контейнеров

Некоторые инструменты могут работать прямо из контейнера:

1. Поместите инструмент в контейнер
2. **Дважды кликните** по инструменту, чтобы активировать его
3. Инструмент будет использовать ресурсы из контейнера вместо рюкзака

> **Совет:** Если у вас в руке предмет из контейнера, нажатие **Tab** автоматически откроет и рюкзак, и контейнер-источник.

Это особенно полезно, когда ресурсы слишком тяжелы для переноски аватаром:
- **Constructor Tool** -- строительство с использованием компонентов из контейнера
- **Destructor Tool** -- разрушенные компоненты попадают в контейнер
- **Spool Tool** -- использование катушек кабеля из контейнера
- **Blueprint Tool** -- размещение транспортных средств в режиме Adventure со всеми необходимыми компонентами в контейнере
- **Rock Rake Tool** -- собранные камни попадают в контейнер

### Ограничения расстояния

Открытые инвентари имеют максимальную дистанцию взаимодействия:
- **Контейнеры**: 100 метров
- **Выброшенные ящики**: 10 метров

При удалении за пределы лимита предметы становятся **серыми** и недоступными для взаимодействия. Окно инвентаря автоматически закроется, если вы отойдёте слишком далеко.

### Крафт из открытых инвентарей

Когда контейнер открыт, система крафта аватара может автоматически брать ингредиенты из него. Это позволяет крафтить предметы, используя ресурсы из контейнеров, без ручного перемещения в рюкзак.

## Размещение компонентов на земле

Некоторые компоненты можно размещать прямо на местности без существующей конструкции.

1. Выберите компонент в руку
2. Наведитесь на землю (в пределах **10 метров**)
3. **ЛКМ** для размещения

| Действие | Описание |
|--------|-------------|
| **ЛКМ** | Разместить компонент |
| **Удерживать Shift + колёсико мыши** | Повернуть компонент |

Компонент автоматически выравнивается по направлению локальной гравитации. Для размещения компонента создаётся новая конструкция.

## Управление окнами

### Закрепление окон инвентаря

Многие окна хранения можно **закрепить**, чтобы они оставались открытыми:
- Контейнеры
- Computer (отсек HDD, IDE)
- Топливные слоты Fission Reactor
- TrashCan
- Выброшенные ящики с предметами

| Действие | Описание |
|--------|-------------|
| **Клик по значку булавки** | Закрепить окно |
| **Клик по X** | Закрыть и открепить окно |
| **ESC** | Скрыть все закреплённые окна |
| **Tab** | Вернуть скрытые закреплённые окна |

### Окна GetInfo

Нажмите **V**, наведя на компонент, чтобы открыть его информационное окно. Эта система работает отдельно от закрепления инвентаря:

| Действие | Описание |
|--------|-------------|
| **Клик по значку булавки** | Оставить окно открытым |
| **Клик по X** | Закрыть и открепить окно |

- Можно закрепить несколько окон GetInfo одновременно
- Закреплённые окна показывают **обновляемые в реальном времени значения**
- **ESC не влияет** на окна GetInfo

Это полезно для мониторинга нескольких компонентов одновременно (например, уровни аккумуляторов, давление в баках, показания датчиков).
