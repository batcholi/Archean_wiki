# Description
Il componente Enrichment Centrifuge separa una miscela di molecole in due flussi distinti in base alla loro massa molecolare.

# Usage
Richiede:
- Un'alimentazione elettrica continua di 1.000 Watt.
- Un ingresso dati per attivarlo.

Il fluido viene iniettato attraverso una porta situata nella parte superiore del componente. In basso, le molecole piu' leggere escono da una porta posizionata piu' in alto, mentre le molecole piu' pesanti escono da una porta piu' bassa direttamente sotto.

Entrambe le porte di uscita devono essere collegate correttamente per garantire il funzionamento corretto.
