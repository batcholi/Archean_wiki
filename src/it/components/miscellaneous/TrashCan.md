# Description
Il TrashCan e' un componente che permette la distruzione istantanea degli oggetti inseriti nel suo inventario.

# Usage
`F` per aprire l'inventario del TrashCan. Una volta aperto, e' possibile depositare oggetti per distruggerli.

>- Il TrashCan puo' essere collegato con un condotto di oggetti per distruggere automaticamente gli oggetti provenienti da una sorgente esterna.
