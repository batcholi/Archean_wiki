# Description
La Small Wheel e' una ruota semplice che puo' solo frenare e sterzare e non ha sospensioni.
Non richiede energia per funzionare.
# Usage
La ruota puo' essere collegata a un computer o altro dispositivo per il controllo.

### List of inputs
|Channel|Function|Range|
|---|---|---|
|0|Brake|0.0 to 1.0|
|1|Steer|-1.0 to +1.0|
