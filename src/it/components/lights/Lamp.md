# Description
La Lamp e' un componente che permette di illuminare un'ampia area con un angolo massimo di 170°. E' particolarmente adatta per essere posizionata a soffitto per illuminare stanze o aree di lavoro.

# Usage
La Lamp deve essere alimentata a bassa tensione e consuma fino a 1000 W a seconda della potenza impostata nel suo menu informativo accessibile tramite il tasto `V`.

I colori della Lamp possono essere modificati tramite il menu informativo della lampada o tramite la sua porta dati.

### List of inputs
|Channel|Function|Range|
|---|---|---|
|0|Off/On|0 or 1|
|1|Red|0 to 255|
|2|Green|0 to 255|
|3|Blue|0 to 255|
