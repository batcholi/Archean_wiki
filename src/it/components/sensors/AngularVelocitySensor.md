# Description
L'Angular Velocity Sensor e' un componente che misura la velocita' angolare su 3 assi (X, Y, Z) in rotazioni al secondo.

# Usage
Una volta posizionato sulla costruzione, il sensore puo' essere collegato a un computer per recuperare la velocita' angolare.

### List of outputs
|Channel|Function|Value|
|---|---|---|
|0|Angular velocity X|rot/s|
|1|Angular velocity Y|rot/s|
|2|Angular velocity Z|rot/s|
