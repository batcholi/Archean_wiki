# Description
Il Temperature Sensor e' un componente che misura la temperatura dell'ambiente in cui e' posizionato.

# Usage
Una volta posizionato sulla costruzione, puo' essere collegato a un computer, ad esempio, per recuperare la temperatura dell'ambiente. La temperatura e' fornita in Kelvin.

### List of outputs
|Channel|Function|Value|
|---|---|---|
|0|Temperature (K)|number|
