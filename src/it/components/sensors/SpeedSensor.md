# Description
Lo Speed Sensor permette di inviare la velocita' relativa all'oggetto genitore (Pianeta, nave madre...) attraverso la sua porta dati.

# Usage
Una volta posizionato sulla costruzione, puo' essere collegato a un computer, ad esempio, per recuperare la velocita' in metri al secondo. L'orientamento dello Speed Sensor non ha alcun impatto sul suo funzionamento.
