# Description
Il Linear Velocity Sensor e' un componente che misura la velocita' lineare su 3 assi (X, Y, Z) in metri al secondo.

# Usage
Una volta posizionato sulla costruzione, il sensore puo' essere collegato a un computer per recuperare la velocita' lineare.

### List of outputs
|Channel|Function|value|
|---|---|---|
|0|Linear Velocity X|m/s|
|1|Linear Velocity Y|m/s|
|2|Linear Velocity Z|m/s|
