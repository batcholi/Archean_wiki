# Description
Il Container e' un componente di stoccaggio con una capacita' di 50 slot.
E' dotato di due porte per collegare condotti di oggetti per ricevere o inviare oggetti.

La porta dati permette di recuperare il contenuto del contenitore come stringa utilizzando il [sistema key-value](/xenoncode/documentation.md#key-value-objects).

>- *Il Container non ha la capacita' di prelevare o spingere direttamente oggetti dalle sue porte, e' esclusivamente un componente di stoccaggio.*
