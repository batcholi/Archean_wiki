# Description
Il Destructor Tool e' uno strumento che, quando equipaggiato, permette al giocatore di eliminare i componenti puntati.

# Usage
`Tieni premuto il tasto destro` per attivare la modalita' di distruzione.
`Tasto sinistro` sul componente per distruggerlo.


---
>- *Non e' possibile distruggere cavi, blocchi o travi con questo strumento.*
>- *Non e' possibile eliminare un sedile se e' occupato da un giocatore.*
>- *Se e' stato aggiunto un [OwnerPad](../components/miscellaneous/OwnerPad.md), e' necessario avere il permesso `Build` per distruggere un componente.*
