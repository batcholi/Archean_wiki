# Description
Il Label Tool e' uno strumento per posizionare piccole targhe sui blocchi per visualizzare informazioni sotto forma di testo.

# Usage
- `Tasto sinistro` su un blocco per posizionare una nuova etichetta.
- Puoi modificare la dimensione e il testo di un'etichetta puntandola e premendo il tasto `V`.
