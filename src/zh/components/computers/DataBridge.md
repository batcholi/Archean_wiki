# Description
Data Bridge 是一个简单的组件，允许将数据端点重新定位到另一个位置。

> Data Bridge 会将数据端点直接重新定位到 Data Bridge 本身。
>
> 为了引用该数据，应通过别名引用 Data Bridge，而不是您尝试读取的组件。
