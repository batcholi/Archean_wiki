# Description
Mini Router 是 [Router](Router.md) 的小型版本，仅有 14 个数据端口。

它需要低压电源才能运行，功耗为 50 瓦。

与标准 Router 一样，它允许为组件分配别名，并且可以与其他 Router 链式连接以扩展网络。

> 与计算机直接通信的 Mini Router 必须通电。链中的其他 Router 不需要电源，可以用作 [Data Bridge](DataBridge.md)，同时仍然能够解析别名和屏幕引用。
