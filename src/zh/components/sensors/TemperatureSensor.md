# Description
Temperature Sensor 是一种测量其所在环境温度的组件。

# Usage
放置在建筑上后，可以连接到计算机等设备以获取环境温度。温度以开尔文（Kelvin）为单位提供。

### List of outputs
|Channel|Function|Value|
|---|---|---|
|0|Temperature (K)|number|
