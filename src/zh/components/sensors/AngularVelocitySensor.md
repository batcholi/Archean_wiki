# Description
Angular Velocity Sensor 是一种测量三个轴（X、Y、Z）上角速度的组件，单位为转/秒。

# Usage
放置在建筑上后，该传感器可以连接到计算机以获取角速度。

### List of outputs
|Channel|Function|Value|
|---|---|---|
|0|Angular velocity X|rot/s|
|1|Angular velocity Y|rot/s|
|2|Angular velocity Z|rot/s|
