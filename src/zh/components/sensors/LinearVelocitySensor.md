# Description
Linear Velocity Sensor 是一种测量三个轴（X、Y、Z）上线速度的组件，单位为米/秒。

# Usage
放置在建筑上后，该传感器可以连接到计算机以获取线速度。

### List of outputs
|Channel|Function|value|
|---|---|---|
|0|Linear Velocity X|m/s|
|1|Linear Velocity Y|m/s|
|2|Linear Velocity Z|m/s|
