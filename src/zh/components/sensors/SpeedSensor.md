# Description
Speed Sensor 允许通过其数据端口发送相对于父对象（行星、母舰等）的相对速度。

# Usage
放置在建筑上后，可以连接到计算机等设备以获取速度（米/秒）。Speed Sensor 的朝向不影响其功能。
