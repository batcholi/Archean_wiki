# Description
TrashCan 是一种可以即时销毁放入其库存中物品的组件。

# Usage
按 `F` 打开 TrashCan 库存。打开后，您可以存入物品以将其销毁。

>- TrashCan 可以通过物品管道连接，自动销毁来自外部来源的物品。
