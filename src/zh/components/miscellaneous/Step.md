# Description
Step 是一种外观为平板的组件，用于创建楼梯。

# Usage
Step 可以放置在方块上，利用其三角形碰撞体（见下图）实现在方块之间的上下移动。

![](StepCollider.png)

此系统允许创建完全自定义的楼梯。

![](StepDemo.png)
