# Description
Enrichment Centrifuge 组件根据分子质量将混合物分离为两股不同的流体。

# Usage
它需要：
- 持续供电 1,000 瓦。
- 数据输入以激活它。

流体通过位于组件最顶部的端口注入。在底部，较轻的分子从位置较高的端口排出，而较重的分子从其正下方位置较低的端口排出。

两个输出端口都必须正确连接以确保正常运行。
