# Description

Power Converter 是一种将高压能量转换为低压能量（或反之）的设备。

此设备非常实用，例如当有可用端口时，可以从高压电池为计算机供电，避免额外添加低压电池。但是，如果您想从低压电池为高压组件供电，总输出功率当然仍受限于低压电池的输出能力。
