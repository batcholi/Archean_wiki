# Description
Destructor Tool 是一个装备后可以让玩家删除目标组件的工具。

# Usage
`按住右键` 激活销毁模式。
`左键点击` 要销毁的组件。


---
>- *你无法使用此工具销毁线缆、方块或梁。*
>- *你无法删除正在被玩家占用的座椅。*
>- *如果已添加 [OwnerPad](../components/miscellaneous/OwnerPad.md)，你必须拥有 `Build` 权限才能销毁组件。*
