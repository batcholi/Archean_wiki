# Description
Label Tool 是一种用于在方块上放置小型标牌的工具，以文本形式显示信息。

# Usage
- `左键点击` 方块以放置一个新标签。
- 你可以通过瞄准标签并按 `V` 键来编辑标签的大小和文本。
