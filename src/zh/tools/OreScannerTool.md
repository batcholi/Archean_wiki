# Description
Ore Scanner Tool 是一个手持设备，可以显示你面朝方向的矿物成分。它显示矿石密度与距离的实时图表，让你在采矿之前定位矿藏。

此工具与 [OreScanner 组件](../components/mining/OreScanner.md)不同——组件是设计用于安装在载具上并通过 XenonCode 控制的，而 Ore Scanner Tool 为玩家提供即时的可视界面。

# Usage

## 基本操作
1. 从背包中装备 Ore Scanner Tool
2. 朝你想扫描的方向看
3. 工具显示该方向上矿石浓度的图表

## 界面
扫描器窗口显示：
- **距离选择器**：选择扫描范围（100m、250m、500m 或 1000m）
- **矿石复选框**：选择在图表上显示哪些矿石
- **图表**：显示矿石密度（Y 轴）与距离（X 轴）的关系

## 矿石类型
扫描器可以检测以下矿石：

|符号|元素|颜色|
|---|---|---|
|Al|铝|银色/浅灰色|
|C|碳|黑色|
|Cr|铬|灰色金属光泽|
|Cu|铜|橙色|
|Au|金|金黄色|
|F|萤石|紫色|
|Fe|铁|锈棕色|
|Pb|铅|深灰色|
|Ni|镍|灰绿色|
|Si|硅|深蓝色|
|Ag|银|亮银色|
|Sn|锡|灰色|
|Ti|钛|蓝色|
|W|钨|深灰色|
|U|铀|绿色|

## 扫描技巧
- 扫描方向基于你视线方向的水平分量
- 当你移动或改变方向时，图表会自动更新
- 选择多种矿石以比较它们的位置
- 铁（Fe）默认被选中，因为它是最常见的资源

---

> **注意：** Ore Scanner Tool 需要在有地形数据的天体上使用。它无法在太空中或没有地形成分数据的环境中运作。
