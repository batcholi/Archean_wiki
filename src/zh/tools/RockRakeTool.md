# Description
RockRake Tool 是一种允许你比手动更快地收集地面岩石的工具。它主要在冒险模式中使用，作为手动收集和采矿钻头之间的中间级别。

# Usage
RockRake Tool 相对于手动收集有两个主要优势：
- 按住 `鼠标左键` 可以持续拾取岩石，而手动一次只能拾取一个。
- `鼠标右键` 可以让你了解目标岩石的成分。
