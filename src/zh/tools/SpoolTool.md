# Description
Spool Tool 用于放置连接组件的线缆和管道，允许在组件之间传输数据、能量、物品或流体。

# Spool Types
按 **C** 打开线轴选择菜单。有 5 种类型的线轴可用：

<p align="center">
  <img src="../consumables/dataspool.png" />
  <img src="../consumables/lowvoltagespool.png" />
  <img src="../consumables/highvoltagespool.png" />
  <img src="../consumables/fluidspool.png" />
  <img src="../consumables/itemconduitspool.png" />
</p>

|类型|颜色|用途|
|---|---|---|
|**Data Cable**|蓝色|连接数据端口进行信息传输|
|**Low Voltage Cable**|红色|连接低压电力端口|
|**High Voltage Cable**|橙色|连接高压电力端口|
|**Fluid Pipe**|灰色|连接流体端口进行液体/气体传输|
|**Item Conduit**|深灰色|连接物品端口进行物品传输|

每种线轴类型只能连接兼容的端口。线轴可以在背包中堆叠，每个线轴物品上会显示剩余长度。

# Usage

## 选择线轴类型
按 **C** 打开线轴选择菜单，选择你要放置的线缆类型。

## 创建线缆（连接两个组件）
1. 瞄准组件的连接器并**左键**开始布线
2. 点击添加中间点以调整线缆路径
3. 瞄准目标连接器并**左键**完成连接

布线过程中：
- **右键**移除最后放置的点（如果没有点则取消）
- **滚轮**循环切换自动寻路的替代路线
- 按住 **Shift** 将线缆吸附到组件表面
- 按住 **X** 将线缆放置在方块/组件的内侧面

## 自动寻路
Spool Tool 具有自动寻路功能，可建议线缆路线。放置时使用**滚轮**循环切换不同的路线排列。

## 创建柔性线缆
要连接不同建筑上的组件：
1. 在一个建筑上开始布线
2. 在另一个建筑的组件上结束

这将创建一条**柔性线缆**，它：
- 在物理上连接两个建筑
- 受物理引擎约束
- 没有力的上限（不会断开）
- 受重力影响

你也可以按住 **X** 在**同一个建筑**的两个组件之间创建柔性线缆。

## 删除线缆
按住**右键**然后快速按**左键**点击已有线缆即可删除。

## 涂装线缆
使用 [Paint Tool](PaintTool.md) 自定义线缆外观：
- 普通涂装更改线缆颜色
- 按住 **Shift** 获得条纹效果
- 按住 **X** 替换所有匹配线缆的颜色
- 同时按住两个键获得透明条纹效果

---

> **提示：**
> - 如果线缆无法创建，可能是你的线轴剩余长度不足
> - 线缆没有与长度相关的传输限制或损耗
> - 线缆不决定传输方向
> - 线缆一旦放置就无法修改——你必须删除并重新创建
> - 柔性线缆比普通线缆更影响性能——尽可能优先使用普通线缆
> - 工具可以通过将工具放入外部容器来使用该容器中的物品
