# Description
Paint Tool 允许你使用自定义颜色和表面效果来涂装方块、线缆、标签和组件。

# Usage

## 控制
| 操作 | 控制 |
|--------|---------|
| 涂装面 | **左键** |
| 涂装整个方块（所有面） | **Shift** + **左键** |
| 从方块取色 | **右键** |
| 在整个建筑中替换颜色 | **X** + **左键** |

## 调色板
Paint Tool 使用调色板系统来保存和组织你的颜色。

### 调色板管理
- **Create**：创建一个新的空调色板
- **Copy**：用新名称复制当前调色板
- **Delete**：删除当前调色板

### 添加颜色
点击调色板中的 **+** 按钮来添加新的颜色槽。

## 颜色选择器
界面底部包含：

1. **RGB 颜色选择器**：使用色调栏和饱和度/明度方块选择任意颜色
2. **Gamma 预览**：显示颜色在你的 gamma 设置下在游戏中的实际外观

## 表面材质
每种颜色可以有不同的表面效果：

| 材质 | 外观 |
|----------|------------|
| **Matte** | 粗糙、无反射表面 |
| **Glossy** | 光滑、有光泽的表面 |
| **Metal** | 粗糙金属表面 |
| **Chrome** | 镜面金属表面 |
| **Transparent** | 透明（用于玻璃效果） |

## 替换颜色
按住 **X** 并**左键**点击任何已涂装的表面，可在整个建筑中替换该颜色。适用于：
- **方块**：替换所有共享相同颜色索引的方块
- **管道/线缆**：替换所有匹配相同颜色的管道
- **组件**：替换所有相同类型和材质的组件

## 对称涂装
当建筑启用了[对称模式](ConstructorTool.md)时，Paint Tool 会自动同时涂装镜像方块。适用于：
- 单面涂装
- 整块涂装（**Shift**）

如果方块位于对称平面上，则会涂装同一方块的镜像面。

> **注意**：替换颜色（**X**）始终影响整个建筑，不受对称影响，因此在该模式下不应用对称。

# Notes
- 涂装方块是**按面**应用颜色的（使用 Shift 涂装所有面）
- 线缆有额外的自定义选项，请参阅 [Spool](../consumables/Spool.md#painting-cables)
