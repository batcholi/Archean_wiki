# Description
Constructor Tool 是 Archean 中用于建造的主要工具。它可以让你创建新的建筑、添加或移除框架，以及放置各种形状的方块。

# Build Modes
按 **C** 键打开径向菜单并选择建造模式：

|模式|描述|
|---|---|
|**Frame**|创建和修改结构框架（钢梁）|
|**Cube**|放置立方体方块|
|**Slope**|放置斜坡/坡道方块|
|**Corner**|放置转角方块|
|**Pyramid**|放置金字塔方块|
|**InvCorner**|放置反转角方块|
|**Triangle**|放置三角形网格面板|
|**Wall / Platform**|放置平面墙壁或地板面板|
|**Strut**|放置细结构钢梁|

# Materials
放置方块（非框架）时，按住 **C** 的同时使用**滚轮**来选择材料。

## Material Masses

| 材料 | 单位质量 |
|----------|---------------|
| Composite | 0.25 kg |
| Aluminium | 0.5 kg |
| Steel | 1 kg |
| Glass | 1 kg |
| Concrete | 10 kg |
| Lead | 150 kg |

> **注意：** 钢梁（框架）每根重 **10 kg**。

### 方块质量计算方式
基本单位为 **25×25×25 cm** 的立方体。方块的总质量取决于：

1. **尺寸**：较大的方块包含更多单位（例如，50×50×50 cm 的方块 = 8 个单位）
2. **形状**：非立方体形状（斜坡、转角、金字塔、反转角）的重量是其等效立方体的 **50%**，因为它们占据的体积更少
3. **材料**：每种材料的单位质量不同（见上表）

**公式：** `Mass = units × shape_factor × material_mass`
- `units` = (size_x) × (size_y) × (size_z)，以 25cm 为增量
- `shape_factor` = 立方体和墙壁为 1.0，斜坡、转角、金字塔和反转角为 0.5。三角形根据其面积有可变的消耗。

### 为什么质量与现实不符
质量经过刻意简化：
- 数值经过**四舍五入**以提高可读性
- 方块并非 100% 实心——它们代表的是带有内部骨架的结构面板，而非实心材料块

# Removing Elements
要移除任何元素（方块、三角形、支撑杆或墙壁），按住**右键**瞄准目标，然后快速按**左键**。这在任何建造模式下都有效。

> 框架的移除有其专属控制方式（见下方 Frame Mode）。

# Usage

## Frame Mode

### 创建新建筑
按住**左键**一秒钟后松开。将创建一个带有单个 3x3x3 金属框架的新建筑。

> **提示：** 创建时按住 **Shift** 可将新框架与地面对齐并设为静态（锚定）。

### 在太空中创建建筑
在太空中靠近现有建筑时，你可以创建一个新建筑，它会生成在**你前方 5 米**处。新建筑将作为最近根建筑的子级创建。这也适用于 [Blueprint Tool](BlueprintTool.md)。

### 添加框架
瞄准现有框架面并快速按**左键**。

### 移除框架
按住**右键**瞄准现有框架面，然后快速按**左键**。

### 添加单独的梁
按住 **Shift** 瞄准现有框架上梁应在的位置，然后快速按**左键**。

### 移除单独的梁
按住 **Shift** 和**右键**瞄准现有框架梁，然后快速按**左键**。

## Block Modes (Cube, Slope, Corner, Pyramid, InvCorner)

### 添加方块
1. 瞄准一个方块或梁
2. 按**左键**放置方块
3. 使用**滚轮**旋转方块（立方体除外）
4. 按住**左键**并拖动来调整大小
5. 按住**左键**的同时使用**滚轮**来调整另一个维度的大小

> **进阶提示：** 拖动/调整大小的平面由你瞄准的方块面法线决定。滚轮沿该法线方向缩放，而拖动则沿其他两个轴缩放。

> **提示：** 在按**左键**之前按住 **Shift** 可以复制你瞄准的方块或逻辑上添加下一个方块。

## Triangle Mode

Triangle 模式允许你放置吸附到建造网格的三角形网格面板。这些面板适合创建弧形、气动面或任何非矩形几何形状。

### 放置三角形
1. 在 **3 个网格点**（现有方块、三角形或支撑杆的顶点）上点击来创建一个三角形
2. 也可以点击一条**现有边**，以 2 个预选顶点开始，然后只需再点击一次
3. 放置时使用**滚轮**来**翻转法线方向**（控制三角形的哪一面朝外）
4. 在轴对齐的方块面上按住 **Shift** 可获得**子网格精度**，吸附到完整的 25 cm 网格
5. 按**右键**可撤销上一个放置的顶点

> 三角形会吸附到方块、其他三角形和支撑杆端点的顶点。每轴最大跨度为 4 m。

### 材料
按住 **C** 并使用**滚轮**选择材料。三角形支持与方块相同的所有材料。

### 平滑着色
瞄准一个三角形时按 **X** 可切换同一组中所有相连三角形的平滑着色。平滑着色使用洪水填充算法应用：它通过**共享边**传播到所有形成连续表面的相邻三角形。

要使平滑着色正常工作：
- 三角形顶点必须在网格上**精确对齐**——相邻三角形需要共享相同的顶点位置
- 不相连的三角形组（没有共享边连接）被视为**独立的平滑组**——切换一个组的平滑不会影响另一个组

> 平滑着色也遵循对称模式。

## Strut Mode

支撑杆是连接两个网格点的细结构钢梁。它们适合创建桁架、天线、脚手架或轻量结构元素。

### 放置支撑杆
1. 在 **2 个网格点**上点击来创建它们之间的支撑杆
2. 支撑杆会吸附到方块、三角形和其他支撑杆的顶点
3. 按**右键**可撤销第一个放置的顶点

> 每轴最大跨度为 4 m。在冒险模式中，支撑杆需要 **Steel Rod** 物品。

### 细分吸附
瞄准现有支撑杆时按住 **Shift** 可吸附到沿其上的细分点。按住 **Shift** 的同时使用**滚轮**来更改细分步长：

|步长|
|---|
|1 m|
|50 cm|
|25 cm（默认）|
|10 cm|
|5 cm|

这允许在现有支撑杆上以精确间隔放置新的支撑杆或三角形。

## Wall / Platform Mode

### 添加墙壁
瞄准框架面并快速按**左键**。

> **提示：** 按住 **Shift** 可使用你瞄准的另一个逻辑框架面。

> **提示：** 放置时按住 **Shift** 可吸附到地面，直接在你前方创建一个静态平台。

# Symmetry Mode
Constructor Tool 支持对称建造，允许你自动建造镜像结构。

可以通过建筑的 **GetInfo** 菜单（在任意框架上按 **V**）启用对称。启用后：
- 所有框架和方块操作都会沿对称轴镜像
- 对称轴位置可以以 **0.125m 精度**调整，允许在方块中心进行镜像放置

# Adventure Mode
在冒险模式中，你需要在背包中准备所需材料：
- 框架需要 **Steel Beams**
- 方块需要**方块物品**（Composite、Concrete、Steel 等）

工具会显示每个操作所需的材料数量。
