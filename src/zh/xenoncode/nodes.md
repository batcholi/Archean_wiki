# 可视节点编辑器
XenonCode IDE 的节点编辑器提供了丰富的节点来帮助你创建脚本和程序。

它足够全面，可以处理复杂的任务，对初学者来说也更容易掌握。如需更高级的功能，你还可以使用文本模式（代码）的 [XenonCode](documentation.md)。

---
## 代码生成

可视节点系统将始终在后台自动生成代码。
当添加 `output`、`print` 或 `execute` 端点节点时会生成代码，它将通过输入依赖关系反向传播。
这意味着添加输入节点不会生成任何代码，但添加输出节点在你将它们连接在一起时将为两个节点生成代码。

---
## 节点列表

以下是编辑器中可用的不同类型节点及其描述（如有必要）：

`Constant Number` 定义一个常量数值。

`Constant Text` 定义一个常量文本值。

`Variable Number` 定义一个可变数值，可以由另一个节点定义和重置。

`Variable Text` 定义一个可变文本值，可以由另一个节点定义和重置。

`Comment` 允许你在脚本中显示注释块。

---
### Input
`By Alias` 通过 Router 通信时使用组件别名的输出节点。

`0,1,2,3...` 使用当前连接到计算机端口上的组件的输出节点。当你将组件连接到端口时，计算机会自动检测到它。

---
### Output
`Print (Console log)` 在计算机控制台中显示数字/文本。

`By Alias` 通过 Router 通信时使用组件别名的输入节点。

`0,1,2,3...` 使用当前连接到计算机端口上的组件的输入节点。当你将组件连接到端口时，计算机会自动检测到它。

---
### Math
Math 节点可以有两个以上的输入，允许你在一个操作中链接多个值。

`Add` 返回两个或更多值的加法结果。

`Subtract` 返回两个或更多值的减法结果。

`Multiply` 返回两个或更多值的乘法结果。

`Divide` 返回两个或更多值的除法结果。

`Absolute` 返回数字的绝对值。

`Negative` 返回数字的负值。

`Modulo` 返回两个数字相除的余数。

`Floor` 返回向下取整到最近整数的值。

`Ceil` 返回向上取整到最近整数的值。

`Round` 返回四舍五入到最近整数的值。

`Fract` 返回数字的小数部分。

`Sign` 返回数字的符号。

`Pow` 返回两个数字的幂。

`Log` 返回数字的对数。

`Sqrt` 返回数字的平方根。

`Clamp` 返回被限制在最小值和最大值之间的数值。

`Step` 如果值小于阈值返回 0，大于阈值返回 1。

`Smoothstep` 返回基于比率的两个值之间的限幅平滑插值。

`Lerp` 返回基于比率的两个值之间的线性插值。

`Sin` 返回角度的正弦值（弧度制）。

`Cos` 返回角度的余弦值（弧度制）。

`Tan` 返回角度的正切值（弧度制）。

`Asin` 返回数字的反正弦角度（弧度制）。

`Acos` 返回数字的反余弦角度（弧度制）。

`Atan` 返回数字的反正切角度（弧度制）。

---
### Boolean
布尔值表示为数值。0 表示 False，其他任何值表示 True。
布尔节点可以有两个以上的输入，允许你在一个操作中链接多个条件。

`And` 如果所有值都求值为 True，返回 1。

`Or` 如果任何一个值求值为 True，返回 1。

`Xor` 如果奇数个值求值为 True，返回 1。

`Not` 返回给定布尔值的相反值。

---
### Compare
`Equal` 比较两个值是否相等，返回布尔值（0 或 1）。

`Greater` 比较一个值是否大于另一个，返回布尔值（0 或 1）。

`Lesser` 比较一个值是否小于另一个，返回布尔值（0 或 1）。

---
### Functions
`Expression` 允许你使用 XenonCode 函数和/或代数编写更高级的数学表达式。

`Switch` 根据输入值返回不同的值。

`Counter` 返回每次调用递增的计数器。可以定义初始值、最小值和最大值。

`Pulse` 当其输入值发生变化时返回 1。

`PulseToggle` 返回一个布尔值，每次输入值变化时在 True/False 之间切换。

`Text` 允许你编写更高级的文本表达式。

`PID Controller` 基于误差、误差积分和误差微分返回 PID 控制值。

`Unit Converter` 在相同类别的常见物理单位之间转换值。支持的类别：角度（Radians、Degrees）、旋转（RPS、RPM）、速度（m/s、km/h、mph、ft/s、knots）、温度（Kelvin、Celsius、Fahrenheit）、压力（Pascal、bar、psi、atm）、力（Newton、lbf、kgf）。

---
### Execution
`Statement` 允许你编写纯 XenonCode 语句，如函数调用。必须输出到 Conditional 或 Execute 节点。

`Conditional` 仅在给定条件求值为 True 时考虑给定的语句。可以与其他 Conditional 链接。

`Execute` 按从上到下的顺序执行连接的语句。

`Include` 在当前脚本中包含另一个 XenonCode 脚本。确保你的变量和常量节点具有唯一名称。

---
### Timing

`Time` 返回当前 Unix 时间戳，以秒为单位，小数部分具有微秒精度。

`Delay` 在 Set 输入从 1 变为 0 后，经过给定的延迟秒数后返回布尔值 1。

`Blinker` 返回在每个定义的时间间隔（秒）交替变化的布尔值 0 和 1。

`Smooth` 在 Start 输入从 1 变为 0 后，返回从前一个目标（或 0）到当前目标值的插值，持续给定的秒数。
