# Installation
## 最低系统要求
- NVIDIA RTX 显卡，并安装最新驱动
- Microsoft Windows 10 或 11，或现代 Linux 发行版

## 购买与游玩
本游戏可在 Steam 上购买。只需在 Steam 商店搜索 "Archean"，或访问我们的 Steam 页面 <https://store.steampowered.com/app/2941660/Archean/>。

购买后，游戏将出现在您的 Steam 游戏库中供您下载。

## 安装
通过 Steam 下载完成后，直接从您的 Steam 游戏库启动游戏。Steam 将自动处理更新和文件管理。

## AMD 显卡
<font color="orange">
Archean 可以在 AMD 显卡上运行（需要 RDNA2 架构或更新的显卡 - RX 6000 系列及以上）。与 NVIDIA 显卡相比，性能可能较低，原因有两点：

- NVIDIA 的 DLSS 技术提供了更好的优化
- NVIDIA 显卡拥有更成熟的光线追踪驱动支持

使用 AMD 显卡时可能会出现一些画面故障和伪影。如果您需要帮助优化 AMD 显卡的设置，欢迎加入我们的 [Discord 社区](https://discord.gg/GeuskE6CmF)获取帮助和更多信息。
</font>

## Intel ARC 显卡
<font color="red">
Intel ARC 显卡与 Archean 不兼容，因为其着色器不支持双精度浮点运算（shaderfloat64）。
</font>
