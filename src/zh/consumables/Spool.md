# Description
不同的线轴允许连接组件以传输数据、能量、物品或流体。

# Usage
## 创建线缆（连接两个组件）
线缆的创建始于和终于组件连接器。

线缆的放置完全自由，允许添加任意数量的点（只要线轴剩余长度足够）来自定义其形状。在创建线缆过程中，如果你放置了多个点，可以使用 `右键` 移除点来调整放置位置，或者在没有更多现有点时完全取消线缆的创建。

- 线缆可以叠放在其他现有线缆和组件上。（要将其吸附到组件表面，你必须按住 `Shift` 键）。
- 你可以按住 `X` 键将线缆放置在方块/组件的内侧面。

## 创建柔性线缆
你可能想临时连接不同建筑的组件，或者简单地将两个建筑连接在一起。
这将有效创建一条柔性线缆。
两个建筑将被有效连接并在物理引擎中受到约束。
没有力的上限，线缆不会断开。
你也可以按住 `X` 在同一个建筑的两个组件之间添加柔性线缆，以获得一条受重力影响的直线（可能略有弯曲）线缆。

## 删除线缆
你可以通过 `右键` 删除线缆，只要你手中的工具是对应的线缆线轴。

## 涂装线缆
你可以使用 [Paint Tool](../tools/PaintTool.md) 像涂装方块一样涂装线缆。
线缆提供两个额外的自定义选项：
- `Shift` 涂装线缆并使其透明
- `X` 正常涂装线缆并带有条纹效果
  
你也可以同时按住两个键来组合两种效果。

---
>- *创建线缆时，如果线缆无法创建，很可能是你当前线轴的可用长度不足以创建该线缆。*
>- *线缆没有与长度相关的传输限制或损耗。*
>- *线缆不决定所传输内容的方向。*
>- *线缆一旦放置就无法修改，你必须将其删除。*
>- *柔性线缆会显著影响性能，而普通线缆即使在一个建筑中有数千条也不会影响性能。尽可能优先使用普通线缆。*
